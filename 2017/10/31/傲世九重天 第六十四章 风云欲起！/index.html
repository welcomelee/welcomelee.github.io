傲世九重天 第六十四章 风云欲起！

发表于 2017-10-31 | 分类于 傲世九重天

这段时间里，上三天表面上风平浪静，但暗地里，却也早已经风起云涌，暗潮澎湃。
厉雄图一回到家族，顿时就成了家族宝贝。甚至厉雄图自己，也万万想不到，自己居然初来咋到就能立即获得如此破格的待遇，看这架势，似乎是将自己当成厉氏家族未来的核心一代来培养一般。
厉雄图茫茫然不知所以然。
回到家族之后，本以为有个管事招待一下安排个清闲地方也就知足了，剩下的以后慢慢努力。
哪想到回来之后居然惊动了三代老祖宗同时出关。
厉雄图虽然心性沉稳，也有些受宠若惊了。
只不过有一点，让厉雄图很不爽，因为包括家主在内，甚至几位老祖宗，对自己的态度，都有些不对劲，怎么说呢，不是亲切，也不是疏远，更加不是排斥，而是一种‘刻意的亲近’！而且，有些人还多少带着一些小心翼翼……厉雄图就想不明白了：这是咋了？我只不过是一个刚从中三天上来的家破人亡的小卒子而已。你们每一个伸出手指头都能捻死我，为啥居然还这样呢？
而且，经常有人问自己一些莫名其妙的话。最普通共同的一点就是：他们都对自己的朋友特别的感兴趣。
厉雄图狂郁闷。这么多年来，自己醉心武学，除了对顾独行董无伤很是钦佩，有些神交之外，哪里有什么朋友？
后来被问得多了，厉雄图本就不是那种肚子里会拐弯的人，干脆直统统的明说：我没什么朋友，更加没有什么过命的兄弟。
那知道这句话出来之后，很多人的态度更暧昧了，居然连连点头：“是是，这是应该的，我也觉得也是，你没朋友，没兄弟，呵呵呵……应该的应该的……”
厉雄图丈二和尚摸不到头脑。
问他们为啥这么问，却一个个都不说。只是含笑看着自己，意味深长……最离谱的是，有一位老祖宗见了自己之后，说道，不愧是……资质的确是不错。
不愧是……什么？厉雄图想破了脑袋也想不出来。
然后偶然听到了别人谈论自己的时候，也是一鳞半爪：厉雄图这小子，真不愧是……呵呵，心机够深的，看那傻呵呵的样子，不知道还以为真的不是哈哈……厉雄图苦苦思索良久，却是越来越糊涂。
然后所有人都敦促自己练功，不遗余力，至于功法、资源更是应有尽有，厉雄图暗暗观察发现，别的家族直系子弟，一般每人每月也就三十块紫晶的份额，就这些已经是天文数字，就连厉拔天，也只是五十块，家族第一公子厉绝，乃是每月八十块，而自己……居然是恐怖的两百块！
比厉绝居然还多出来将近两倍！
而且居然没有人提出异议，更没有人提出不服……就连那年轻子弟之中号称是第一人的厉绝，那可是见了老祖宗都是不卑不亢的人物，对同辈更是毫不假以辞色，但见了自己，却也依然是笑脸相迎。看不出多热情，但绝对没有任何的敌意，还有一种淡淡的修好之意……厉雄图这种性格的人到后来都有些害怕了：这……这到底咋回事啊……于是，他就在这样的纳闷中，开启了自己在上三天的日子，不得不说，厉雄图资质的确超凡，又有庞大的家族在后面全力支撑，一旦静下心来修炼，功力修为突飞猛进……而他的修为进步的越快，大家都是含笑很欣慰的样子，丝毫不感觉诧异，一般来说，到了皇座，三年突破一品就算快的，若是有人一年之中突破一品，那就是天才，绝对会引起一片震惊。
但厉雄图仗着体内一股莫名的力量，半个月提升了一品修为！这本应该是惊世骇俗的事情，但，厉氏家族的前辈们居然都一副理所当然的样子……这就更让厉雄图不解了，咋回事啊？我靠要闷死我咩？
不过最近一段时间，来找他聊天的少了，而且每一个人都是行色匆匆脸色沉重，似乎发生了什么大事，隐隐约约听的说是各大家族之间与厉家的摩擦和冲突渐渐的多了起来……厉雄图问时，所有人都是笑着对自己说不用担心，你绝对没事的……我靠我担心什么啊？我能有什么事啊………………诸葛世家。
第五小院。
第五轻柔坐在院子里的凉亭里，手中拿着一卷书，神色间一派悠闲。面前一杯香茗，袅袅升腾茶香。
微风徐来，竹林摇曳，刷刷作响，满院清净，一片潇湘……一人无声无息的出现在他身后，良久，第五轻柔似乎没有半点发觉，凝神看书，淡淡微笑。
这人终于沉不住气，道：“好自在。”
第五轻柔回身，微笑：“原来是七哥。”
来人正是诸葛家族这一辈人之中的老七，诸葛云雾。也是诸葛家族之中，与第五轻柔性格最是相合，最为投契的一个。
其实，只是与第五轻柔投契这一句话，就可以看得出来，这个诸葛云雾，绝对不是一个简单的人物！
若是一般庸俗之辈，第五轻柔怎么会与他投契？
“轻柔，九大家族现在都已经是风起云涌，眼看着就要战祸连绵九重天了，你却还是在这里逍遥自在。”诸葛云雾一身布衣，神情恬淡，面带微笑，两眼细长，长眉入鬓。面白无须，看上去就似乎是一个世外之人，对于人间之事，似乎根本不在心中。
他纵然在说着九重天的局势，也是云淡风轻，如同清风过耳。
“纵然战火连绵九重天，但诸葛家族只要有七哥在，也是万无一失！”第五轻柔微微笑道：“不过……七哥这一次来，定然有事情。轻柔正洗耳恭听。”
“嗯，不错，七哥是无事不登三宝殿，既然来到你这个智囊家中，自然是有事情无法取决。”诸葛云雾轻轻一笑。
“请七哥示下。”第五轻柔终于将手中书本合上，抬起头看着诸葛云雾。
“打开天窗说亮话，轻柔，我知道你的本事，此来只是问你一个答案。”诸葛云雾很直接的说道。
“答案……”第五轻柔似乎才到了对方要问什么，对方还没问，他的眉头就皱了起来。
诸葛云雾在他面前坐下，轻声说道：“现在家族正在打算，逐步与厉氏家族加大磨擦力量，试图引起冲突，揭开那一层……”
第五轻柔淡淡的‘哦’了一声。
诸葛云雾吸了一口气，道：“轻柔，你定然知道，这乃是因为……九劫剑主……”
第五轻柔蹙了蹙眉头，没有说话。
诸葛云雾看着第五轻柔蹙起的眉头，眼中亮光闪了一下，沉默的道：“厉氏家族，最近接到上三天来的……厉雄图，轻柔，这个人，你怎么看？”
诸葛云雾双目如星，凝视着第五轻柔的脸色，道：“厉雄图，现年二十四岁。中三天厉家人，自幼资质出众，三岁就有根基，五岁冲破武士，九岁冲破武师，十三岁到武宗，十七岁，破武尊，十九岁，到王座，二十三岁，王座四品。但，到上三天之前，亡命湖一战中，他二十四岁，居然是皇座二品巅峰，而且战力足可完胜普通皇座四品！甚至，能追平皇座五品上！”
第五轻柔嗯了一声，道：“你是说，他在一年之内，或者是不到一年的时间，从王座四品直接提升到皇座二品巅峰的事情？可疑？”
诸葛云雾点点头，看着他，静了一会，似乎是专门留出时间让他思考，然后才道：“最近传出消息，厉雄图自从到了上三天，这不过半个月功夫，已经提升到了皇座三品，中。”
第五轻柔眉毛动了一下：“有这等事？”
诸葛云雾淡淡一笑：“这速度，挺快的吧？”
第五轻柔没有说话，沉思起来。
“还有，在临到上三天的时候，有人送他一粒丹药。那时他伤重垂死，但服用之后，立即全然恢复。”诸葛云雾加重了语气：“而且那人的一招，便是……屠尽天下又何妨！”
“那一粒丹药，九成九，就是九重丹！除了九重丹之外，普天之下，再也没有任何药，能有这种效果！”诸葛云雾说到‘九重丹’这三个字的时候，分明再次加重了口气，而且，声音有一种激动。
第五轻柔淡淡的笑了笑，不置可否。
“当时，九大世家的人与执法者走在一起，彼此监督，几乎没有离开过。那人似乎是一直没有找到机会，所以……才在临进入上三天的时候，冒险赠药，交手一招，立即远遁无踪！就连执法者，也追之不及。”
诸葛云雾轻声道：“在这件事上，唯有一个疑点。那就是，天空中出现了异象，而那人若真的是九劫剑主，就应该出现在那现出异象的地方，而不应该是去赠药！”
“但，据回来的人描述那一招的威势，以及变化手段，九大家族中几位老祖宗一致认定，那，就是九劫剑主的九劫剑法之中，最为凌厉的一招，屠尽天下又何妨！而且，绝对没错！”
