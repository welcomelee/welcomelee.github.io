傲世九重天 第六十六章 为何练刀？

发表于 2017-10-31 | 分类于 傲世九重天

第五轻柔皱着眉头，苦苦思索：“如果这个人与他不是同一个人……那么，在这世间，我就不只是一个可堪匹配的对手！迂回的手段虽然巧妙，但直接的阴谋却更加可怕！而且令我感到了威胁和恐惧。”
“究竟是不是？”
第五轻柔百思不得其解，他站在这地图前面，竟然呆呆的站了一上午。
他这个小院，属于第五家族，在诸葛家族之外，第五家族依附于诸葛家族，多少年来，才在最近数百年获得了这么一小块的地方。
而第五轻柔由于在第五家族之中属于嫡系，所以拥有这一个小院。与其他人不同的是，在这个小院之中，没有任何一名仆人。
侍女书童，统统没有。这里，完全属于第五轻柔自己！在这里，第五轻柔不敢相信任何人。
所以他虽然呆呆的站了一上午，也没有任何人打搅。良久，他才颤抖了一下，醒过神来，默默道：“此事虽不可解，不过……相信定有原因，只是，我一时没有想到而已。”
“所有人都认为，九劫剑主现在在中三天，上不来。所以大家现在的防备，也并没有真正地森严壁垒……”
第五轻柔轻轻笑了笑，退后几步，手指伸出，轻轻落在地图东南方向，眼神凝重而意味深长。
萧家的地盘。
“楚家，就在这里。楚家，只是一个小家族。”第五轻柔凝神看着地图上，只有米粒大小的那么一块地方，喃喃道：“平沙岭。”
第五轻柔静静地看着这米粒般大小的地方，眼神有些奇怪。良久，喃喃道：“九重天入口关闭之前，最后一批进入的人，便是楚家的大少爷，带着儿子……听说他儿子失踪了很多年。这一次终于找回来了……”
他歪了歪头，有些失笑，喃喃道：“听说这位楚家大少爷的儿子名字叫做楚阳……真的太有趣了……”
他的神色更加迷惑了起来：“按照天地异象来看，九劫剑主是在参悟九劫的时候，引发了异象。而那个地方，距离上三天入口还有很远……”
“他既然在参悟九劫，就不可能出现在世人面前。而楚家大少找儿子，也不会是将一个正在参悟的人拉过来就认准儿子，然后立即回返上三天，这分明来不及……”
“若是九劫剑主在中三天，楚阳在上三天……”第五轻柔呻吟了一声，感觉自己的脑袋要爆炸了：“这是怎么回事？这到底是怎么回事呢？……”
九大家族从来不将各自家族势力范围之内的小家族放在眼中。在九大家族掌权者眼中，这些小家族，实在是没有被重视的必要。
九大家族都不放在眼中，执法者自然更加不会放在眼中。
所以，就连九重天封闭之最后的通行记录，执法者除了记录了一下九大家族中在外之人之外，对于楚飞凌带着儿子回归，只是语焉不详的记载了一笔：楚家楚飞凌携妻带子回归。
就这么一句话。
所有人看过之后也就忘了。
唯有第五轻柔，在发现了这个消息之后，被这个‘楚’字引起了敏感神经，专门秘密的查了一下。
才发现，这‘带子回归’，居然是带的失踪了十八年的儿子。
然后第五轻柔再次打听，才终于打探出来：楚家找到的这个失踪的后人，名字叫楚阳！
看到‘楚阳’这两个字的时候，第五轻柔清楚的记得，自己手中的茶杯啪的落在了地上，当时自己的震惊，无法言喻。然后第五轻柔就将这件事，死死的埋在了自己心里！
甚至，心中还有些兴奋——或者，第五家族的希望，就要真正开始？
但九劫剑主被封锁在下面的消息，却再次让第五轻柔迷惑了起来。
“要不要走一趟呢？”
第五轻柔抬起头，看着诸葛家族密地的方向，想到那里，还有一件极端重要的事情在等着自己，拖了这么久，还一直没有下文来到，不由得皱起眉头，喃喃道：“等这件事情结束，我就亲身前往东南。”
………………在另一个地方，也有两个人苦恼的在抓头皮。
董无伤，芮不通。
两人跟随执法者来到上三天，已经超过了半月。
执法者并没有食言，也的确是为董无伤介绍了一位刀圣老师。但这位刀圣老师却在见了董无伤一面之后，立即闭关了。
原因就是，董无伤与他见面之后，曾经进行了几句话的谈话。
这段谈话，本来应该是刀圣大人考校董无伤的，但……结果却出人预料。
刀圣：“你为何而练刀？”
董无伤：“因为我喜欢刀。”
刀圣沉默。
董无伤默然。
刀圣：“刀在哪里？”
董无伤：“刀在手中。”
“不在心中？”
“不在。”
“心中无刀，如何练刀？”
“心中有刀，刀主心。手中有刀，才是人主刀！”
…………谈话进行到这里，就结束了。因为刀圣大人接着就陷入了沉思，然后就宣布了闭关。并有些怨怼、有些惋惜、有些后悔、有些怅然的……留下一句话：“我不配当他师傅。”
刀圣大人的问话，其实就是一个个的刀道领悟。为何练刀？不管你说为了天下为了苍生为了杀人或者为了守护……都是符合刀圣大人的刀道领悟的。
但董无伤是一个简单的人，只是用了一句‘因为我喜欢’就将这一切，在懵懂之中全堵了回去。
是啊，任何事情，都比不上这三个字：我喜欢！
若是不喜欢，就算明白告诉你，你练好了这个，天下就是你的，你也不会去练。因为不喜欢！
自己不喜欢，你打死我也没用。
所以董无伤最简单的回答，反而是最返璞归真的领悟。刀圣不仅没达到目的，反而等于是被对方教训了一顿——虽然董无伤本身并没有这样的意思。
但刀圣大人却分明觉得对方是在训斥自己：你早已经偏离了当初练刀的方向！练刀，不是为了杀人，不是为了守护，也不是为了天下苍生。因为你开始练刀的时候，才只有两三岁。
两三岁的孩子，知道什么杀人守护天下苍生？
从若干武器之中，选了刀，只是因为一份本能的喜欢，仅此而已！随着年龄的增长，逐渐的为自己的选择加上了太多的光环。
但，当你向着这些光环前进的时候，不管是高尚的还是崇高的……但无一例外的都是：你已经不是为了那份单纯的喜欢而练刀！
你已经背弃了自己！尽管你已经变得伟大……所以刀圣立即沉默。
接下来，本想扳回一局，便问刀在哪里？董无伤又是最简简单单的来了一句：刀在手中……然后一个延伸，刀圣大人就成了刀的奴隶。
刀圣大人颜面全无，丢脸的同时，却也是被当头棒喝，从此闭关。
执法者惊。
董无伤告辞时，执法者本不欲放他走，但那位闭关的刀圣居然因为此事废除刚开始一天半的闭关，也要出来为董无伤说情。
“放他走！自由的刀，才是刀。若是强行留他，刀，则断！”
董无伤没有拜成师傅，却多了一个忘年交。虽然这忘年交说完这句话，立即又消失了……于是董无伤带着芮不通，墨泪儿，一路出了执法堂。
但出来之后，董无伤就立即知道，自己的选择有些孟浪了。
因为在上三天，金银根本不值钱。值钱的，是晶石。
但三人手上，最缺的就是这东西。带着金子银子银票金票，却发现自己居然是一文不名！这种感觉很是奇怪。
但这时候，就显出了莫天机的算计。
莫天机岂能不了解董无伤的脾气？所以他刻意的为董无伤搭配了一个助手：芮不通。
芮不通一直不显山不露水，但……芮不通却有一个显赫的身份：这位天兵阁的六爷，乃是神偷鬼盗的传人。
神偷鬼盗是干什么的？
所以芮不通实在是有一手神鬼莫测的偷盗技术。所以，只要有芮不通在这里，那么，就算是穷光蛋，也没关系。因为……下一刻就有了。
事实上，莫天机给出的这三人搭配虽然是临时起意，但却实在是天衣无缝。
初来咋到，啥也没有，咋办？偷！
一旦有所成就，立稳脚跟，又有一个出类拔萃的内政领袖人才：墨泪儿。这可是黑魔，偌大的杀手组织人家都能管理的井井有条，你董无伤一点儿家底，还给您管理不好啦？
一旦开疆扩土，开始发展，墨泪儿摇身一变，又是统帅型人才。
而财政大臣，自然是芮不通无疑：谁能从偷儿手里偷走东西？
芮不通的吝啬与小气贪财，在兄弟们之间可是出了名的……其实仅次于傲邪云。
傲邪云乃是天生的吝啬，芮不通是后天的培养……所以这三个人在一起，董无伤唯一的任务就是：猛打猛冲！只要你能冲，后面就不断地有人将你的屁股擦得干干净净的！
你能创下多么大的家业，就有人为你管理多么大的家业！
所以，三个人只是惆怅了还不到半个时辰，芮不通就出去逛了一圈，回来的时候，怀里多了二三十块瓦蓝瓦蓝的蓝晶，还有百十块白生生的白晶。
熬了三天，董四爷不好意思老是让芮不通去偷，于是乎自己蒙上脸，带着兄弟和老婆直接去入室抢劫了………………
