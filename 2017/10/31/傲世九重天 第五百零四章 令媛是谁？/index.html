傲世九重天 第五百零四章 令媛是谁？

发表于 2017-10-31 | 分类于 傲世九重天

“这居然是……灵酒！”蔚公子大吃一惊，急忙运功，引导这突如其来的强大修为归纳入经脉，片刻之后，蔚公子振衣起身。
看着自己浑身的污垢，哭笑不得。
果然是好东西，刚才这三口，足足让自己增加了三百年修为！只是瞬间，自己的修为居然已经到了至尊四品的突破临界点！
这简直就是不可思议的事情……既然这个水壶中的酒如此神庙，那么，另一个里面装的是什么？
蔚公子突然极度的好奇起来。
试着拿起来，打开盖子，一闻。
蔚公子一屁股坐在了地上，突然两眼睁开到了最大限度，两手死死地抱着紫晶水壶，呻吟一般的道：“生命之泉！不对……乃是指差一丝就能进化到生命之泉的后期的生灵之泉……我的天哪……”
他失魂落魄的坐着，眼中神色，却是不可置信的狂喜。
良久，才醒过神来，随即就赶紧盖住，顾不得自己身上臭气四溢，就向着楚阳远去的方向追了过去。
“楚阳，你给我站住！你你你……你他么的……你从那里找来的……”
蔚公子的大呼小叫响彻了中三天。
疯了…………楚阳一路狂风一般奔驰，中途改变了方向就去了傲家。
蔚公子一路狂呼，却是背道而驰。
双方距离越来越远。
这正是楚阳的心眼：哥现在正急着与众兄弟团聚，你丫不要捣乱了。要不然，我还不如留下看你的震惊脸色呢，肯定很过瘾……只是现在，你再急也不如我急啊。
哥的时间真心的不多了。
至于为啥让蔚公子看到生灵之泉；那是因为……嘿嘿嘿……这货修为增长了这么多，不做金牌打手实在是太可惜了啊啊啊…………楚阳到来的时候，只见傲氏家族之内很是热闹。
大家都是步履匆匆，傲氏家族似乎有很多的客人到来。
楚阳狐疑的抓住一个问了问：“怎么了？怎么这么热闹？”
这家伙很不耐烦：“你丫是不是中三天的人啊？这么大的事情都不知道。”
“这么大的事情？”楚阳一喜：“难道傲邪云他们出关了？”
“不是！”这傲家的武士很是不满：“你难道不知道傲家主要做寿了？各大家族都来祝寿！”
“原来如此。”楚阳嘴角抽了抽，心道看来自己是赶得真巧，看样子是非要被宰一刀了。
正在这时，远方车队车琳琳马萧萧；声势浩大前来。上空一面大旗迎风飘扬：“顾”！
顾家！
“看到了没：顾家少奶奶兼大小姐亲自前来了！”傲家武士很高傲很光荣的说道。
“顾家少奶奶兼大小姐？”楚阳被这个称呼雷了一下。半晌才反应过来：这就是那位传说之中的顾妙龄了。
顾独行的心上人。
楚阳闻名已久，今日才是第一次见面，不由得大是好奇。
只见傲氏家族大门猛地打开，里面有人快步迎了出来。
居然是几个妇人；一个个雍容华贵，那武士吓了一跳：“家主夫人亲自出迎了，这真是……给面子！”
楚阳咳了一声；明白了，顾家来的女眷，傲天行作为家主却是男人，出迎不合适，把自己老婆搬出来了。
果然，顾家马车车帘一掀，一个袅袅婷婷的人影出现了，远远地就见到她快步上前行礼，然后，一干女人们就欢欢乐乐亲亲热热的走了进去。
随即，大道上又有车队前来。罗家的，董家的；谢家的；莫家的，纪家，相继到来。
谢家的来认识谢丹琼的夫人和四个小妾；罗家来人是罗克武；董家的来人是董无泪；纪家当然是纪铸。至于莫家的，则是前任家主带着莫天机的两个小妾前来。
楚阳顿时明白了。
这兄弟们几个在这里闭关，家里有女人的岂能不照顾一下相思之苦？心爱的人一去不回，这么长时间不见面，谁不牵挂？而且现在几家关系这么近，家族高手暗中保护，多多走动，也是好事儿。
这些女人们能够等到傲天行大寿才商量好了一般前来，已经是相当的克制着是相思之苦了。
楚阳有些感喟。
人同此心，心同此理。
自己常年在外，莫轻舞，铁补天她们，岂非比她们还要可怜？
在这一刻，楚阳下了一个决定：以后若是有把握，就尽量带着一起出行，若是没有把握，那么，自己决不能离家太久！
……这些兄弟中，基本上有女人的都来了，至于没有女人的，则只能是男人们来了。
楚阳很纳闷儿看着面前的武士：“我也不算是祝寿的，还抓住你问东问西，你怎么这么放心都跟我说？”
那武士很不屑的看了看他，鼻孔朝天：“每年……像你这样的傻冒，都有不少。再说了，现在在中三天，谁敢招惹我们傲氏家族？就你……还想捣乱？”
楚阳一头黑线。
这时，莫氏家族的队伍过来，傲氏家族门前，傲天行早已经站立等候。莫星辰虽然已经退居幕后，但其确实与自己同一辈分的人物，这个面子不给可不成。
热情地迎接了莫星辰，傲天行转头与莫星辰把臂同行；两个老狐狸一辈子没怎么对付，但此刻各自的儿子却是生死之交，也突然间亲热了起来。
两人走了几步，傲天行突然咦了一声。
转头却是远远的看到了楚阳。
不由一阵大喜！
快步走了过来。
那位武士见到家主大踏步走来，吓得小腿肚子几乎转了筋：“都怪你都怪你……拉着我聊天，现在可倒好，被家主看见了……看来我这顿棍子是少不了了……”
楚阳哭笑不得。
“楚阳，你怎地在这里？”傲天行惊喜之极，一双眼珠子瞪得老大：“既然来了怎么不提前说一声？老夫也好亲自出来迎接于你。”
楚阳笑了笑，拱拱手：“不知道是伯父大人寿诞，冒昧的来了……此刻正尴尬。”
“这有什么尴尬？”傲天行一把拉住他胳膊，拖着就走：“来到了这里还不进家门，简直是将老夫当做了外人……真真是气杀我也。”
说着气煞我也，但口气却是兴奋快乐亲近的要了命。
原先那位武士目瞪口呆的看着楚阳被家主大人从自己身边拉走，目瞪口呆。楚阳？这个名字怎么这么眼熟？
是了，天兵阁的老大岂不就是这个名字？难道刚才跟自己聊了半天的这个青年人，居然是目前中三天仅次于暗竹的大势力天兵阁的阁主大人？
自家少主和其他几家少主们共同的老大：楚阳？！
这么一想，顿时两条腿就弹起了琵琶；哎哟喂我的老大啊……您说您这么一尊大神，跟俺这么一个小虾米逗了半天趣，如今吓得我尿都要出来了……却见傲天行一转头，狠狠看着自己：“你这狗头，楚世兄来了居然也不通报，就这么晾着贵客！该当何罪？回去自己去领八百棍！”
八百棍？
这位仁兄两腿一软就跪了下去。家族刑堂那些人，八棍子就能打烂一位王座；八百棍……真是能打的可以包饺子了。
“算了，他也不知道，还多亏了他为我介绍呢。”楚阳微笑：“此人有功啊。”
“有功？！好！”傲天行大笑：“起来吧，楚世兄说你有功，既然如此，一会儿去账房领八百两银子！作为对你的犒赏！”
那位仁兄晕晕乎乎的一时间竟然迷惘了……前一刻八百棍子就要打在了身上；后一刻就成了赏了八百两银子……这种从地狱直线提升到天堂的遭遇，让这位武士很悲催的脑筋半天没转过弯。
那边，傲天行已经亲热的拉着楚阳走了。
莫星辰这还是第一次见到楚阳。
楚阳到莫家的时候，莫星辰正是心情极度懊恼，闭关谁也不见，一切都交给了莫天机。自然是没有见到的。
“莫伯父。”楚阳亲切的笑笑，主动打招呼。不管以前怎么样，这货毕竟是莫轻舞的亲生父亲，自己未来的老丈人。
此刻留下一个好印象还是必要的。
果然，莫星辰顿时受宠若惊又有些心虚的大笑起来：“你就是楚阳？哈哈哈……果然是一表人才！果然是少年英雄！名不虚传，名不虚传呀。”
楚阳谦虚的笑：“您老过奖了。”
莫星辰亲热的打量，他当然知道楚阳乃是自己女儿莫轻舞的牵挂的人，心中想起以前的事，有些尴尬，有些没话找话，鬼使神差的问道：“嗯，楚世兄，你……家中可有妻室了？”
楚阳愕然：“您老这话的意思是……”
“若是你还没找媳妇儿，老夫还有一个独女……未曾出阁。”莫星辰哈哈大笑：“届时，你们可以多多的，接触接触哈哈……”
一边的傲天行撇撇嘴。
这老货，想得倒美，殊不知上次楚阳来的时候，老夫就说过这个问题，可是被无情拒绝了。如今，就等着你这老货碰一鼻子灰吧。
只见楚阳眼睛一亮：“令媛芳名是？”
莫星辰哈哈一笑：“小女你也认识，她的名字，叫做轻舞；若是楚世兄有意；小女现在芳龄十四，长得貌美可爱，正在从师学艺……”
