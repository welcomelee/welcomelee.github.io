傲世九重天 第一百七十九章 这是为什么呢？

发表于 2017-10-31 | 分类于 傲世九重天

这一战，足足持续了一天半！
楚阳到后来已经打得有些支撑不住了。
就在感觉自己快要修为枯竭的时候，白衣人突然大叫一声，跳出战圈，怒道：“果然有些本事！你等着！”
突然身形拔起，当空一个盘旋，消失得无影无踪！
楚阳兀自拿着长剑又划了两下，才终于止住，晕头转向的转了两个圈子，一屁股坐倒在地上，大口大口喘气，大笑道：“真痛快！”
这一战，虽然一直是守多攻少，但却实实在在的是打的酣畅淋漓，浑身解数都使了出来，所有潜力，都挖掘了出来。
柔水剑意，经过这一战，已经无限的趋于成熟！
而且，道境之中的领悟，让楚阳领会了许多。
甚至，楚阳心中隐隐在盼望着，这个白衣人再来一次，再这样的战斗一次！
楚阳渴望这样的战斗。
因为，他的修为提升太快，从皇座九品，到剑帝四品，刷的一声就上来了。中间的历练实在是乏善可陈。
而这一战，他一直用剑帝三品的境界战斗，不知不觉的，剑帝三品所有的领悟，在这一战之中，都已经融会贯通，更兼有道境的力量，更加的领悟深入了……楚阳呼呼喘气。
魏无颜走了过来：“怎么样？”
“爽！”楚阳哈哈一笑，道：“这一战，打得真是酣畅淋漓！身心都非常的愉快！”
魏无颜无语的说道：“道境啊道境啊。”
突发奇想：“你休息休息，跟我打一架。”
看到楚阳三战之中，每一战都进入了道境，魏无颜突发奇想：会不会是在这黑血丛林之中，只要战斗就能进入道境？
顿时心痒难熬。
楚阳也很有兴趣，道：“我休息休息，就跟你打！”
…………半晌之后。
“来吧。”魏无颜跃跃欲试。
“好！接招！”楚阳大笑一声。
半晌之后。
魏无颜郁闷了。
因为，楚阳又进入了道境！而自己，却是根本没有……“这是咋回事？”魏无颜百思不得其解。
“再来！”
“好！”
…………“郁闷死我了……再来！”
“呃……好。”
…………“咋回事？你都进去了五次了，怎么我一次也没有？”魏无颜怒了：“再来！”
“汗。”
……“再来！”魏无颜咬牙切齿两眼发红。
“大爷您就饶了我吧。”楚阳瘫在地上，四脚朝天，脸色青白，手脚颤抖：“累死我了……我不行了……”
魏无颜不依不饶：“男人不可以说不行，起来！与我再战！”
楚阳浑身抽搐：“男人对女人不可以说不行，可你是男人啊……实在没兴趣了……”
魏无颜仰天长叹：“我想进入一次道境，咋就这么难呢？”
楚阳无语的道：“你都是半步至尊了，你说你压着修为跟我打，能有什么领悟？怎么可能进入道境？”
说者无心，听者有意，魏无颜浑身一震，顿时茅塞顿开：“不错！定是如此！”
也不仅仅是他，远远的地方，一个白影也是浑身一震：不错！定是如此！
楚阳躺在地上，有些韵味无穷的说道：“只可惜，刚才那白衣人只能应付我的剑帝三品，若是我将修为提到四品，他可能就支持不住了……有点儿可惜。”
魏无颜鄙夷的道：“能进入一次道境整整一天半，你就知足吧……还想着借助人家继续突破？你想的也忒美了一些。”
楚阳哈哈大笑。
遥远的暗影中，那白影动了动，喃喃自语：“他能够继续突破……？”突然若有所思。
三人休息了好久，才离开了这一片狼藉的地方。楚阳拽出一个水囊，咕咚咕咚直接喝干了。
刚才战斗出汗，让楚剑主几乎脱水……三人又开始了一天前的行程：魏无颜一边背着楚乐儿赶路，一边冥思苦想。而楚阳则继续双目放光的搜寻灵药；这一日，走出了五十里。
眼见的前面越来越是阴森，越来越是黑暗。已经快要接近中心地带。
“小贼！还我的朱果树来！”一道白影从天而降，一巴掌狠狠拍向楚阳。
楚阳早有准备：“来得好！”
举剑上迎，两人在一瞬间就是翻翻滚滚打成了一团。
这一次，白衣人的修为似乎有了大幅增进，居然隐隐到了君级两三品的样子，将楚阳的三品剑帝压的直接喘不过气来。
楚阳将心一横，将修为在刹那间提升一层，剑光如惊鸿射出。
“四品剑帝！”白衣人似乎有些兴奋：“你也提升了！”
楚阳哼哼冷笑：“怎么，不行么？”
“提升了又如何？你照样还是要死在我手上！”白衣人冷笑不绝。
“未必！”楚阳提升至四品剑帝，柔水之中已经开始带着狂涛的声音，如风雷涌动：“且看到底是谁死在谁的手里！”
白衣人不再说话，只是加紧了进攻。
楚阳全神贯注，见招拆招；白衣人的招数滑溜之极，根本不能把握，楚阳一开始还想着万一杀了他很可惜，但逐渐的物我两忘，全神沉浸在战斗中去了，忘了这一切……一边的魏无颜拽着头发，长吁短叹：道境！又是道境！
这小子又进去了！
与他同样郁闷的还有另一个人：白衣人。
白衣人看着楚阳呼吸悠长，身上散发着那种玄妙的道境气息，也是气不打一处来：这小子怎么进入道境这么容易？
他加快了进攻速度，但楚阳也随之增强了剑光强度。
自始至终，昨日的战斗再度重演：三七！依然是三七！精确到了一丝一毫都没有变化的三七分成！
楚阳三成攻势，白衣人七成！
白衣人依然是压着楚阳在战斗，而且，战斗到中途，白衣人竟然闭上了眼睛，完全凭着感觉，在与道境之中的楚阳在战斗。
而且，若是有人能够看透他的伪装的话，就会发现他的脸上的表情，似乎在沉重的思索着。
这样的表情，与旁边的魏无颜的表情毫无二致。
这一次战斗，又是一天一夜！
到后来，楚阳已经是大汗淋漓。禁不住就想提升自己的修为到五品剑帝的层次进行战斗……就在这时，白衣人一声长笑：“明日再战！明日决不让你好过！”身形一起，白影一闪，旋风一般拔起，消失得无影无踪。
楚阳呼呼喘气。
心中却突然的升起奇怪的心情：这个白衣人，究竟是何用意？
怎么看他的战斗方式，倒像是纯粹在帮我稳固修为，领悟剑术一般，这个人……真的有这么好心么？
但若不是，他是为什么呢？
楚阳思索着，但楚大少很快就将这种疑问扔到了九霄云外：不管他有什么目的，但他却不会消失，明天还会来……无论如何，到最后他都会说的。
而且这样的战斗对我有无限的益处，多打几次又有啥关系？
目前来说，最重要的还是找灵药！灵药越多，等到我走出这黑血丛林的时候，底气也就越足啊。
剑灵偷药固然可行，但在九大世家这种非常了解九劫剑主的大家族之中，难保不会被人认出来或者怀疑。
哪怕只是有一点点的怀疑，自己的隐藏也就变得全无意义。
所以，还是从黑血丛林这种天赐之地寻找要好一些。
所以，在这次战斗之后，楚剑主寻找灵药的步伐非但没有停止减慢，反而更加的疯狂了！
从方圆百里扩展到了方圆五百里，而且，除了前面的中心地带之外，似乎还有越来越扩大的趋势，向着方圆一千里去发展。
这样的恐怖的扫荡，让黑血丛林之中的强者们一个个都是两眼流泪：忒狠了……这直接就是在刮地皮啊……再说了，哥们，哪怕你只收集顶端的也行；可是这位爷居然连十年八年的也不放过。剜到篮子里就是菜，多多益善……这就更让人无语了……你还让我们活不活了？妈的，以后受了伤想在黑血丛林里面找点儿金疮药都没了……但上峰有令，不准自己等人对付这小子，要安安静静的放他过去……所以大家也只有干瞪眼，一个个的狂吞唾沫看着。
“哇！”楚阳终于忍不住发出一声惊喜的叫声。
这一声叫，让暗中的黑血丛林强者们心脏都是为之一阵抽搐：又发现宝贝了！而且绝对是好宝贝……这小子这段时间被黑血丛林惯坏了，一般的宝贝已经根本不能让他惊讶了……楚阳在一片阴暗之处，发现了一个乳白色的小尖尖。
这个乳白色的小尖尖，让楚阳大喜过望！
因为这东西他认得。玉雪灵参！
发现了这东西，连九劫空间之中的剑灵也是翻了两个跟头来表示兴奋。
这里可是黑血丛林之中啊。
土地坚硬成这样子，而且这玉雪灵参长在黑血丛林，居然还是白色的……这得生长多少年啊？
最起码，三五千年是长不成这样子的！
而且，在黑血丛林的三五千年……可不是外界的三五千年可以比拟的啊。这样的东西，绝对是整个九重天也是可遇而不可求的！
纵然是楚阳的定力，也是忍不住心头剧烈的跳动了一下！
然后他才仔细地、小心的走过去。
