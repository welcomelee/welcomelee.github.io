傲世九重天 第九百零九章 什么是猪？

发表于 2017-10-31 | 分类于 傲世九重天

董无伤猝不及防，即便已臻至至尊九品仍旧承受不起，顿时有如元宝一般被打飞了出去，依然有些丈二和尚摸不到头脑：“到底是怎么哄……哎哟喂……痛……”
“我叫你怎么轰！轰！”墨泪儿满脸通红的飞身而上，追着董无伤往后飞的身子，一连串的打击过去，愤愤怒骂：“就是这么轰！这么轰！这么……这么这么……轰！”
一边，正在高兴得找不到北的所有兄弟一时间不知道发生了什么事情，顿时全静了下来。
这是什么情况？！
一个个的看着墨泪儿大发雌威，董无伤如同沙包一般被打来打去，轰轰的声音不绝于耳，董无伤惨叫的声音此起彼伏，不由得一个个都是一头冷汗，面面相觑。
这……咋回事？
兄弟们啊……找媳妇儿一定不能找这样的，真滴会死人的啊……这是典型的前车之鉴啊，我的个老天爷啊，这也太狠了，幸亏老董已经是九品至尊了，这修为要是差一点，还不被轰成肉泥？！……尤其是纪墨，看着董无伤在那里挨揍，不由得想起了呼延傲波，刹那间就是浑身一哆嗦，当真是感同身受，竟有些忍不住的提心吊胆起来——貌似呼延傲波那妞儿……也不比墨泪儿和善多少，其狂暴之处，甚至犹有过之，老董这身板都这样了，我呢………我的妈呀，越想越恐怖，心里越发的恐慌，纪二爷强自镇定，却仍旧是张口结舌，喉咙里不由自主的咽着唾沫，眼睛都有些发直了——吓得。
貌似两条腿也有些颤抖。
君之今日，恰似我之明天啊……呜呜……今天这事貌似又创造了一宗超级记录，一个区区至尊六品，把一个九品至尊中级强者打得变猪头，全无还手之力，更把另外一个观战的九品至尊吓得腿发抖，还将另外六七位九品至尊吓得噤若寒蝉……墨泪儿大小姐这份成就，相信不但是空前，还将绝后了，旷古绝今，古今一人！
“我让你怎么轰！”伴随着墨泪儿一声恼羞成怒的大喝，董无伤的硕大身子犹如流星一般被踢飞了出去，轰轰轰地撞断了好几棵大树，最终，整个人嵌在了最后一棵大树树身里面。
眼看到墨泪儿转头气冲冲走来，兄弟们几个人都是很识趣的转过头，若无其事的寒暄起来。这个时候问怎么了？问为什么？都是触霉头滴，很可能惹祸上身，谁也不傻！
“呀，嗷嗷，你现在发型可真帅啊。”
“是嘛哈哈哈……纪墨你现在也很英俊，真的很英俊。”
“嗯嗯，小狼现在可是很有性格啊。”
“顾独行今天穿了一身黑衣服哇，真酷……”
“就是就是，莫天机居然跟傻逼似的笑了……”
……诸如此类没有任何营养的话，就在那七位九品至尊之间很热烈的说着。
看着这帮无耻没有下限的人，墨泪儿恨恨的跺了跺脚，几乎想要再冲回去将董无伤再揍一顿，这口气要是出不干净，自己憋屈。“姐，你到底咋了？”莫轻舞凑在墨泪儿耳朵边上轻声问道：“无伤哥哥可不像是会惹你生气的人啊，是不是有什么误会……”
“误会？那有什么误会！这个混蛋！”墨泪儿兀自恨恨不已：“他耳聋了！就该揍！”
莫轻舞瞪大了俏丽的眼睛。耳聋？
九品至尊会耳聋？
这个……只是就算真耳聋了，也不至于揍啊，貌似两者完全不挨边吧！
董无伤呻吟着从树洞里挣扎出来，一瘸一拐的往回走，之前那股子威猛霸道的凌厉气势早已经烟消云散了，取而代之的是满头雾水，以及一种‘丈二和尚摸不到头脑的无辜与冤枉’；很郁闷的道：“你这到底咋了？这是咋了？我怎么着你了，你……”
一听这话，墨泪儿顿时又要飞身而上。
这夯货，还能更气人一点不？
能木能？
莫天机急忙出面打圆场：“好了好了，无伤，我有很重要的话要跟你说，过来下，赶紧的。”
莫天机是何等人精？！头脑自然非是董无伤可比。
虽然这两人并未明说事情始末，但一转头就想得出来。墨泪儿定然是因为大家都坐着火箭一般的提升了而她自己没有提升还在原地踏步心里自然是不平衡了……不患贫，却患不均！
这本就是无可厚非的事情，事实上，相信换做任何人心里也都会不平衡的。
更何况墨泪儿当初在中三天与这里所有人尽都是平起平坐的超级人物，一路走来，也不曾稍落半步，然而此刻一下子落后了那么多，心中能好受才怪。
偏偏董无伤一个榆木疙瘩脑袋而且还是处在提升之余的极度兴奋之中什么都不想的榆木疙瘩脑袋……对于墨泪儿跟他说话竟然没有听清楚，若是不被修理，那才叫怪了。
“咋了？啥重要的话不能当面说，还要背着大家，你赶紧说啊！我还有话要跟泪说呢……”董无伤走过来，低声问道。
“哎……”莫天机说话之前先叹了口气：“我现在总算明白了什么叫做恨铁不成钢……你怎么这么二，你真以为我跟你有话说啊？！我跟你老实说，你老婆分明是因为……”
说着低声解释了一顿。
“哦~~”董无伤恍然大悟。终于明白了；赶紧转头就奔了回去。
“哎……我话还没说完呢……”莫天机话还未说完，这货已经急吼吼的走了，不由得一阵瞠目结舌。我还没有教你如何哄好……你急什么？
只见董无伤一路小跑来到墨泪儿身前，挠着头一脸无奈，道：“老婆，这事儿真没办法，你别着急，也别上火，更不要感到不平衡，虽然大家都提升了，就你没有提升，但这实在是这就是没办法的事儿啊……”
话没说完，墨泪儿已经彻彻底底无地自容的飞出一脚。
这次是真真正正、羞愤难当的全力施为——轰！
董无伤再一次变作空中飞人。比之前飞得还远，撞倒了更多的大树……在半空中，董无伤脸上更加的迷惘了……我真的已经完全理解了，她怎么还打我？这咋回事呢？女人真不好伺候啊，不理解要打，理解还打……墨泪儿捂着脸跺着脚，无地自容的怒吼：“董无伤！你这头猪！！”
“真是猪啊！”其他兄弟们同时叹息，摇头，一脸无语：“真正已经是猪到一定地步了……”
本来这事儿是一件挺**的事，墨泪儿无论如何也是不好意思当面说出来，但董无伤这一解释，却是弄得天下皆知，把所有事都放到了明面上……你让墨泪儿一个女孩子脸上如何下得了台……噗！
董无伤再度撞倒了好几棵大树，在数十丈之外站起来，挠了挠头，一脸的迷惑，歪着头沉思了好久，感觉自己貌似没什么地方做错了，才突然感到冤枉之极的开口：“为什么？？！为什么又打我？！”
一听见这句问话。
包括墨泪儿和莫轻舞在内，所有人动作整齐划一，尽都用手捂住了脸。
真没想到……这货居然到现在依然没有明白过来……这得有多么多么的天赋异禀啊……千万不要再用猪来比喻这货，这明明就是埋汰人家猪！
这次，对于董无伤表现出来的诧异与冤枉，大家都很默契地选择了袖手旁观，有几个热心人还想要上前解释，却被纪墨和罗克敌联手阻止：开玩笑，你们解释什么？咱们大家看看戏多好啊……纪二爷甚至暗中开了赌盘：赌，董二爷到底能二到什么时候才能醒悟过来……于是众兄弟下注者纷纷而上……“哎，真真是一群唯恐天下不乱的家伙，兄弟有难，不说帮把手，还要落井下石，真不是东西……”莫天机看着这样的现象，不由无奈的摇了摇头，然后走过去，下注一百块紫晶…………“接下来，咱们要怎么办呢？”谢丹琼眼中有无限的跃跃欲试。
大伙提升了修为，自然要找个对手练练啊，如此才能巩固现阶段的修为。
“还是等楚阳上来之后再说。”莫天机沉稳的点点头，示意大家稍安勿躁。
一听这话，大家都安静了下来，是啊，楚老大现在还没上来呢，大家提升如斯，却不知道楚老大现在又提升到什么地步了？
“谁跟我赌？”纪二爷赌性大发，那边赌盘尚未结束，居然又要开盘：“我赌楚老大现在至少也是九品高级了。”
“有三个选择可供下注：九品中级，九品高级，九品巅峰。”
“下注的速来，此地人傻钱多，这便宜不占白不占了啊。”罗克敌放声大叫，随即下注五百紫晶在九品中级。
根据大家的经验，楚老大一般情况下，只要是提升，大抵都跟兄弟们的水准差不多。再说现在只是九劫剑第八截，还没到最后一节呢，应该跟大家是一样的程度，绝对不会高出太多，所以选择九品中级是非常有苗头的。
正是人同此心，心同此理。
大家纷纷下注。大多都是赌中级。
唯有莫轻舞本着对楚阳有着无限相信，直接下注九品巅峰五百紫晶。纪墨眉花眼笑：“还是轻舞妹妹够意思，是看我穷，故意给我送紫晶补亏空来啦……好吧，看在小丫头一片真心，下注九品巅峰的一律一赔十！”
