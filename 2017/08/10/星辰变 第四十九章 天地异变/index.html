星辰变 第四十九章 天地异变

发表于 2017-08-10 | 分类于 星辰变

今曰第三章！
——————完全由镜子构成的府邸，在端木玉刚刚抵达的时候，便自然开启了。这反而让秦羽、端木玉、周显一批人都沉默了。
诡异！
这府邸这么多年没开启过，为何现在自动开启呢？巧妙到极点，就诡异了。诡异地让场上众人，明明看到了府邸通道，一时间却都不敢进去。
“秦羽兄，这府邸怎么回事，我刚刚到，它怎么就开启了？”端木玉对于情况也不太熟悉。
秦羽皱眉说道：“在你来之前，这府邸无论如何也开启不了。周显他还在我之前到来呢。而端木兄你一到，这门便自动开启了。”
“哼，有什么好怕的？不就是一座府邸吗？”周显忽然低喝一声，便悍然地朝这镜子府邸通道走去。
端木玉轻笑一声：“这周显倒是不惧，也对……他拥有空间神器，危险时候藏入空间神器，静等他的父皇救援，的确是安全的很。”
“只要不是神王亲自出手，便没什么危险。”端木玉看向秦羽，“秦羽兄，我们进去吧？”
秦羽沉吟着。
秦羽自己非常的清楚，这镜子府邸的背后，正是有一名神王藏着呢。否则……那‘易风’为何一直用神识观察着？
端木玉、周显等人察觉不了，秦羽却察觉地清楚。
“不过，为了幻灵镜。”秦羽双眉凝结在一起，为了夺得第二个名额，秦羽也顾不得那么多了。
“就是神王，在我空间之力覆盖区域，也休想伤得了我兄弟。”秦羽对自己的空间之力很自信。
秦羽对端木玉微微一笑：“好，我们也进去。”
“可不能让那周显抢先了。”端木玉笑着说道，同时整个人如同离弦之箭，闪电般窜入了镜子府邸中。
秦羽三兄弟也同样极速窜入了镜子府邸中。
可当秦羽三兄弟刚刚窜入府邸中的时候，只听得“轰隆隆”的声音再度响起，那扇大门竟然关闭了起来。
秦羽三兄弟不由身形停下，前方的端木玉三人也朝后面看来。
“情况越来越有意思了，这易风……到底要干什么？”秦羽心中思虑着。
“大哥，情况似乎不妙。”黑羽低声说道。
秦羽当即传音道：“小黑，费费放心，你们二人就在我身边，如若有什么危险，我会将你们直接送入新宇宙中。”
侯费、黑羽也都点头。
“我们走。”秦羽便继续前进。
这海底府邸，从外面看是一面面镜子构成，可是从内部看，墙壁等都是黑漆漆的，在通道的一些位置，还有一些明珠，放出幽幽的碧绿色光芒。
秦羽三兄弟速度飞快，飞速窜行在府邸中。
一会儿，便来到了体积最大的后殿。这后殿，占地广阔的很，可是后殿的布置却极为的凌乱。
镜子！
无数的镜子！
广阔的后殿中，地面上随意的放着无数面镜子，有的镜子体积只有巴掌大小，有的镜子却有数十米高，镜子数量之多，绝对过万。
如此多镜子，让后殿中不少人都傻眼了。
“这么多镜子，哪一面又是幻灵镜？”侯费嘀咕着。
侯费嘀咕的问题，此刻也困扰着端木玉、周显等人，他们也在仔细观察着一面面镜子，可是他们对幻灵镜知道的太少，也不知道该如何分别。
秦羽却是站在后殿中央，一动不动。
“这里的镜子，共有九万九千九百九十九面，每一面镜子都差不多，任何一面镜子，都没有不同于其他镜子的地方。”秦羽空间之力观察之下，很容易就判断了出来。
秦羽需要的，是不同之处。
真正的幻灵镜，肯定不同于其他的镜子，可是秦羽的空间之力也发现不了。
整个府邸并不算多大，前殿等区域还是比较小的，秦羽的空间之力也观察过了，那些地方根本没有镜子。
唯有后殿，才有大量的镜子。
“这里的镜子数目，也非常特殊，九万九百九十九，这个数字，又代表什么呢？”秦羽心中思索着。
有空间之力，秦羽根本不需要如周显、端木玉等人那般辛苦的寻找。
侯费、黑羽二人却是帮着秦羽，仔细地观察一面又一面镜子，黑羽掉过头看向秦羽，传音道：“大哥，这镜子如何辨别呢，好像没什么不一样的。”
忽然——“小双！”
只听得一声叫声从远处传来，秦羽的空间之力一下子发现了：“端木玉怎么了？好像受到了影响。”
“费费，小黑，你们在我身边不要乱动。”秦羽感到了一阵不妙。
“神识，灵魂之力？”秦羽一下子发现了玄奥所在，此刻有着一道道的神识波纹从各个方向渗透入端木玉的脑海。
“那些神识波纹强度很弱，甚至连下部天神的神识强度也没有，绝对攻击不了人。可是那端木玉，为何整个站在那一动不动？难道，是因为幻灵镜？”秦羽知道，这幻灵镜能够探知人心灵，甚至于能够让人快乐或者痛苦等等。
“难道幻灵镜就在端木玉身旁？”秦羽的空间之力立即仔细观察。
可就在这时候——秦羽感到，那一道道的神识波纹，陡然覆盖了整个后殿。和进入端木玉脑海同样的神识波纹。
“费费，小黑，小心一点。”秦羽嘱托一声。
秦羽倒是不太担心，这种强度的神识波纹，就是不防御，单单以上部天神灵魂元婴的强度，也不是这种神识所能伤害的。
那神识波纹，瞬间，覆盖了整个后殿。
同样，秦羽的脑海，也渗透入了不少神识波纹。
******秦羽感到整个人所看到的场景陡然变幻了，原本的后殿消失了，出现在秦羽眼前的是北极圣皇殿。
北极圣皇殿上方，十三位神王齐聚。
“好真实的场景，这是幻境吗？”秦羽自己都有种难以判断的感觉，身为阵法宗师，秦羽此刻却无法判断了。
一切都是那么真实。
可是，秦羽的灵魂还联系着‘新宇宙’。自然还联系着那空间之力，秦羽心中知道……只要自己愿意，完全可以瞬间脱离这种状态，恢复正常。
“脱离？”
秦羽心中竟然有了舍不得的感觉。
“诸位。”北极圣皇姜梵高居于大殿之上，眼前的姜梵，无论是气势，还是笑容等等，都是那么真实。
“如今三个名额都已经定下来了，秦羽他连续夺得三个名额，也就是说……毋庸置疑的，秦羽成为了我的女婿，立儿的丈夫！”
听着上面的北极圣皇姜梵的宣布，秦羽整个人都感到激动。
……一切都是那么真实，随后众人的恭维，周显的怒而离场。
在秦羽亲自炼制的府邸中，秦羽、立儿举行了盛大的婚礼，大量的人齐聚在这里，众人庆贺，秦羽也是兴奋的喝众人推杯换盏。
多久了？
一切不是为了今曰吗？
……洞房之夜。
“立儿。”掀开头盖，和立儿彼此凝实。
“秦羽大哥。”立儿眼中噙着泪花，泪花中闪烁的是幸福。
秦羽心底很清楚，这一切都不是真的。可是……秦羽却非常享受这场景，他舍不得破坏这种情况。
这种状态下，时光好像流逝的很快。
和立儿在一起快乐的曰子过了很久很久，他们两人后来也有了自己的孩子，那是一个男孩。
“羽哥。”躺在床上，立儿满脸慈母光辉，怀中抱着一个婴儿，“我们的孩子叫什么呢？你起个名字吧。”
“我们当初分隔那么久，彼此一直牵挂思念着，恩……这个孩子就叫秦思吧。”秦羽爱怜看着立儿以及那可爱的婴儿。
这一刻……秦羽心中涌动的，是无尽的疼惜关切。
忽然——场景陡然消散，等场景再次形成，竟然又出现了北极圣皇殿。
“诸位，三个名额，两个名额归秦羽所得，还有一个名额归周显所得。经过我和夫人最后的斟酌选择，我们决定，周显成为我们的女婿，立儿的夫君。”
秦羽很冷静。
因为秦羽知道这是假的场景，只是秦羽心中有种疼痛的感觉。
“幻灵镜，果真奇妙无双，明知是假的，可是看到这种场景，依旧有种愤怒的感觉。”秦羽心中感叹。
这事情很简单。
自己，即使知道某个场景是假的。可是当场景跟真的一模一样，你自己也无法察觉的时候……发生的一切依旧会让自己心中震颤。
“周显想要跟立儿结婚，结婚？不，我不可能让这一切发生，绝对不可能！”秦羽看着事情进展，竟然有着一种‘毁天灭地’，杀死一切阻碍之人的癫狂。
“哼！”
秦羽冷哼一声，灵魂元婴的灵魂之力一阵震荡，眼前一切场景顿时崩溃了。
……眼前的场景，又恢复成了海底府邸的后殿。秦羽清晰地看到，侯费、黑羽等人都陷入那种奇特状态中。
“那还是幻境吗？”秦羽心中感叹着。
“这一切，唯有那奇特的‘幻灵镜’才会做到吧。的确没有什么攻击力，只要愿意，可以瞬间脱离那种状态。可是……也可以靠它，享受没有实现的目标理想。”秦羽心中感叹。
达到上部天神的境界，一般的幻境根本影响不了他们。
而‘幻灵镜’却能够让他们感受到最快乐，或者痛苦的事情。而且那种感受，是非常真实的，源自灵魂的！
“陷入幻灵镜影响中，好像很长，实际上才过了盏茶功夫。”秦羽心中一算便知晓了。
“呼！”黑羽也睁开了眼睛。
“大哥，刚才那是怎么回事？实在太真实了！”黑羽脸上也有着惊叹，“大哥，你知道吗？刚才，我竟然看到了我的父亲和母亲，我只有在梦中才看到父亲和母亲的样子，可是刚才，我那么真实的和父亲母亲在一起。”黑羽眼睛都有些湿润了。
秦羽安慰地拍了拍黑羽的肩膀。
“只是一些悲惨的场景，我实在承受不了，所以就离开了那种状态。”黑羽感叹一声。
“我也是，我比你更承受不了。”秦羽笑着说道。
随后的时间里，侯费、周显等一个个人接连清醒了过来。可是随着清醒的人越来越多，所有人都惊异起来。
“这端木玉怎么还没有清醒过来？”侯费惊异说道。
秦羽也摇了摇头。
“端木玉他和我谈过他的感情，只是我想，他跟说的不过是最简略的，真实的情况可能比我想的复杂吧。此刻，他陷入的也很深。”秦羽低声感叹说道。
忽然秦羽脸色一变。
这海底府邸并不大，秦羽的空间之力不但覆盖整个府邸，还渗透了出去，外面的海底场景也在观察之中。
可是这个时候——只见以海底府邸为中心，一个巨大的漩涡诞生了，同时整个南野群岛周围的海域完全翻滚了起来。
那数百座岛屿的天神们，都布置出禁制抵挡咆哮的海洋。
“轰隆隆~~~”天空中雷电疯狂闪烁。
同时，各种颜色的云朵聚集在上空，一共七色，七色祥云不断聚集着，雷电如同游蛇一样在七色祥云中游荡着。
秦羽的空间之力只能观察到海底的剧变，无法观察到天空的变化。
海底府邸中的秦羽等众人也感到海底府邸处于震颤中。
“呼！”很是突兀的，整个海底府邸忽然消失了，秦羽等一群人亲眼看到……那诺大一个镜子府邸，竟然飞了起来，同时极速的缩小，一会儿变成了巴掌一块镜子，落在了一人手中。
这人，正是易风。
而易风，目光却是聚集在端木玉身上。
