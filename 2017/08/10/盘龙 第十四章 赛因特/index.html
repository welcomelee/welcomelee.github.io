盘龙 第十四章 赛因特

发表于 2017-08-10 | 分类于 盘龙

这光头黑甲壮汉瞥了一眼乔治，而后便去休息了。他很满意让乔治担任管家，一是乔治是一个下位神，没有什么威胁。二是乔治的确有才干，将他这一山脉人马管理地是井井有条。他如今都认为，当年没杀乔治，是很明智的选择。
这次死的人不多，乔治很快便安排好了。
夜，山脉城堡内，乔治站在窗户前，透过窗户可以看到那一弯红色妖月，这一弯红色妖月在提醒着他，这是冥界！不是他的家乡玉兰大陆。
“这样的日子，还要持续下去。”乔治心中暗道，“也不知道这一千多年，玉兰大陆到底怎么样了。按照那位叫奥丁的人，霸道的行为。恐怕耶鲁老大，乃至龙血城堡，都要遭受到威胁。”
乔治是一个很冷静的人，不管遇到什么样的环境，都能很快适应。
在冥界中，有一个人皆共知的事情——亡灵们都是不知晓前世记忆的，只有当他们蜕变成神的那一刻，灵魂受到天地法则洗礼，那一刻尘封的前世记忆也会尽皆知晓。
乔治死亡的时候，便是圣域。
他的灵魂吸入冥界后化为亡灵，亡灵实力是由灵魂决定。乔治的灵魂是圣域灵魂，他转化为亡灵后，也是圣域亡灵。当初是圣域亡灵的时候，乔治没前世记忆，只知道杀死别的亡灵，吞噬别的亡灵灵魂，让自己变强！
强大自己，让灵魂不断强大。然后逐步地领悟……终于，他突破了，成神了！
成神的一刻，也才知晓前世记忆。
知道他过去的名字——乔治。
作为当年玉兰帝国的柱石大臣，乔治在为人处世中可比林雷强的太多，少年时期在恩斯特魔法学院，乔治便擅长结交朋友，在玉兰帝国官场上也是如鱼得水。如今进入这强盗窝里。
或许修炼、战斗，强盗们擅长。可在管理方面，他们比乔治差的远了，乔治担当这管家，也是令大家心服口服。
“现在第一目的，是将财富积累到，足以在城内购买房子。”乔治心里早有计划，“等那时，就能在城内安全地修炼下去！不必在这浪费精力。”乔治身为这一山脉的管家，地位之高，仅仅在这一脉首领之下。他明明可以弄到中位神神格炼化，可他不想。
他要独立成神！
在这，他虽然不敢明目张胆捞钱，可以他的能力，悄无声息中便能将大量钱财弄到自己空间戒指里。别人怎么可能发现？
等钱够了，谁还呆在这强盗窝里？
“可惜，老三他在地狱，想见他也难。”乔治暗叹。
*******北骸府范围的确太大，林雷乘坐金属生命，也是飞行了数月时间，才抵达海德城外的大草原上。
“这北骸府主住的地方，就好像一个城池了。”贝贝透过窗户朝外看去，赞道，“单单府兵驻扎的人就有数十万，一眼看过去，尽皆都是成排的房子，还真有那么一番气势。中间最高的那栋城堡，估计就是府主住处了。”
林雷也看过去，那最高的城堡，足有百米高，通体为象牙白色。
“希望那北骸府主没出去。”林雷感叹一声，“如果这北骸府主不在，那我可就白跑一趟了。”当即收起金属生命，林雷、贝贝二人直接朝远处飞了过去。
北骸府主居处，自然戒备森严，大量府兵巡逻着。
“什么人！”林雷他们距离老远，便听到远处府兵传来大喝声。
林雷却是和贝贝继续飞过去，那些府兵们立即有数十人一同迎上来，为首的一个额头有着尖角的府兵喝道：“这可是府主居住的地方，外人不能擅自闯入，你们二人还是赶紧离去。”
“请去向北骸府主禀报，就说四神兽家族长老‘林雷’，来拜见他。”林雷淡笑说道。
“你说你是四神兽家族长老，就是了？”为首府兵嗤笑道。
林雷淡然一笑，体表弥漫开一道土黄色光芒，直接将那为首的府兵覆盖，那府兵措手不及下，也不由身体一颤，直接歪倒在地上。而后咬牙才勉强站立起来。
“相信了吧？”林雷淡笑道，同时也收回土黄色光芒。
“相信，相信。”那府兵连道。
他完全被吓住了，就通过那重力空间他已经敢肯定，眼前人绝对是达到七星恶魔层次的人物。“两位请在这等一会儿……我先去禀报。”这独角府兵说了一声，当即掉头便直接飞向那最高的城堡。
“你家府主，现在可在？”林雷向其他府兵们询问道。
那些府兵们也明白，眼前这二人不是一般人，其中一大胡子府兵笑道：“大人，我们也就是普通的府兵，府主大人如果离开，估计我们都不知道。等一会儿，队长他回来了，就知道了。”
林雷只能静静等了，贝贝则是喃喃道：“希望那个府主可别出去哦。”
片刻——“林雷大人，林雷大人。”那通报的府兵老远便立即喊了起来，身形一闪便过来了，脸上满是笑容，“府主大人知道林雷大人你来非常的高兴，已经命人开始准备宴席了。还请林雷大人随我一起进去吧。”
林雷听了不由和贝贝相视一眼。
“这么热情？”贝贝喃喃道。
“这是好事。”林雷一笑，当即飞向前，“前面带路！”
在赶往那城堡的途中，贝贝疑惑神识传音道：“老大，这北骸府主，知道你来。似乎很客气。难道他听说过你？老大你的大名，都从地狱传到冥界了么？”
“见了就知道了。”林雷此刻心中很愉悦，“还有，等会儿见到了那北骸府主，你可别在那闹腾。我们是求人办事的。”
“知道。大不了我不开口。”贝贝摸摸鼻子。
林雷不由笑了笑，这一次来冥界办事似乎很顺利，轻易地就能见到那北骸府主。
“林雷大人，到了。”那府兵站在城堡门前。
林雷已经看到不少侍女正捧着餐盘朝城堡里面松了。当即林雷、贝贝二人进入了这城堡内部。
……距离府主城堡大概数百米外，还有一座略微矮些的城堡，此刻一名穿着灰色长袍的男子正站在阳台上随意地欣赏着景色，如果林雷看到他，肯定一眼认出，这位正是他做梦都想杀死的敌人——巴达位面监狱的邪恶王者‘奥丁’。
奥丁当年来到冥界后，也是在亡灵圣山传送阵出现。
他第一站，便来到北骸府，在北骸府中度过一段时间他也明白了，以他的实力，在冥界中虽然算是超级高手，可是还没有达到真正巅峰。无法和冥界府主比。在这过程中，他也碰到了北骸府主。
和北骸府主的几次比试，他输的心服口服，也就成了北骸府主的手下。成为北骸府主麾下第三使者。
“恩？”奥丁惊异看向远处。
只见两个人影，在府兵带领下进入了府主城堡。
“是他们！”奥丁瞳孔陡然一缩，面色大变，“这林雷怎么来冥界了？”他奥丁怎么都不可能忘记，他的风系神分身在玉兰大陆位面，所遭受到的屈辱。虽然说那屈辱只能算是初级……可对奥丁而言，却是他这辈子受到的最大屈辱。
他恨林雷！
当时更是放狠话，让林雷来冥界找他。
“没想到这林雷还真跑来冥界了！看来，不杀死我他不死心啊。”奥丁目光发寒，“现在竟然敢跑到府主这，这林雷是知道我在这，还是因为其他事情？”奥丁有些怀疑了，如果林雷知道他在这。
敢光明正大过来见府主？
须知，他奥丁可是府主的属下。
“本来我想就这么算了。谁想，你还追到冥界来。哼。”奥丁气的当即身形一闪，直接飞出了自己城堡，朝城主城堡飞过去。
当初风系神分身，已经完全感受到林雷实力了。奥丁明白……论物质攻击，林雷远超他。就是灵魂攻击，林雷也当初更是能够让他奥丁陷入浑浑噩噩状态，失去意识。奥丁通过这一点就肯定——林雷的灵魂方面，不比他弱！
“要杀这林雷，我自己可不行。只能求府主帮忙了。”奥丁当即从侧门，进入府主城堡。那守卫也知道奥丁身份，也不阻拦。
*******客厅内。
长条形餐桌上摆满了各种美食，一瓶瓶珍贵名酒也放好。林雷和贝贝并而坐在一旁，在对面是一名蓝袍雍容的女人，而主座上则是坐着一袭白袍的俊美中年人，这中年人面白无须，笑地眼睛总是眯着。
这中年人，正是北骸府主！
“林雷，我虽然在冥界，可也听说了青龙一族林雷长老的大名。今天林雷你来我这，真是一件让人开心的事情。”这中年人声音柔和，“我来介绍一下，这位是我的夫人阿妮塔！哦，我都忘记自我介绍了，我赛因特！”
“赛因特先生，阿妮塔夫人。”林雷微笑道，“这位是我兄弟贝贝。”贝贝立即挤出一丝笑容。
林雷心中还是很高兴，看现在这北骸府主的反应，应该比较好说话。
“刚听说林雷你的消息不久，你就来我这了，真是让人惊喜，不知道林雷你这次来，有什么事情？”赛因特笑得眯起了眼睛，“有什么事情尽管说，只要我赛因特能帮上忙的，一定会帮忙。”
林雷心中一松，旁边的贝贝立即笑了起来：“你肯定能帮上忙的，你是主神使者对吧。”
赛因特一怔，旋即笑着点头。
“赛因特先生，主要我有非常重要的事情，想要拜见冥界主神。可是……我根本不知道冥界主神在哪里。还请赛因特先生指点一下，我去哪里，能够见到冥界主神。不一定要见七个中某一个。任何一个都可以。”林雷连说道。
“要见冥界主神？”赛因特大吃一惊，旁边他的夫人也惊讶看向林雷二人。
“能告诉我，你见冥界主神什么事情吗？”赛因特询问道。
林雷略微迟疑。
赛因特立即笑了起来：“我只是问问。恩……你想去冥界主神，这事情的确有些麻烦。我也知道冥界主神在哪里。可是你即使去了，主神如果不愿意见你，你去也没用的。主神那可是高高在上的。”
林雷心中大喜，这赛因特果然知道。
“赛因特先生，还请你告诉我主神在哪里。至于主神是否见我，就看我运气了。”林雷连道。
赛因特迟疑了下，微微点头。
就在这时候——“府主！”一道声音直接在赛因特脑海中响起，赛因特不由眉头一皱，也神识传音回应道：“奥丁，你有什么事情？”
