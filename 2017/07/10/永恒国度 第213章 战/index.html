永恒国度 第213章 战

发表于 2017-07-10 | 分类于 永恒国度

咔嚓！！
不过，就在此刻，在传送门中，再次出现一道身穿银白法长袍的男子，法杖挥舞间，赫然只看到，一道银白色的刀刃毫无征兆的朝着易天行当头劈斩下去。在刀刃中，能感受到一种无法想象的锋芒，这种锋芒，在一出现，就让易天行浑身汗毛都开始倒竖起来。
鸡皮疙瘩都冒出来了。
魔法——次元斩！！
一种来自空间的锋芒，让人几乎能感受到一种强烈的死亡气息。那是一种不可抵挡的绝望，那锋芒，实在是太可怕了。可怕到要任由它斩在身上。
叮！！
易天行的反应不可谓不快，几乎在次元斩出现的瞬间，已经本能的将手中的战矛调转矛锋，直接一矛砸向次元斩，一阵清脆的响声中，次元斩中蕴含的锋芒，甚至连矛意都被斩破，那锋芒，那力量，太过可怕，就算这样，这一矛依旧将次元斩挡了下来。
直接崩碎次元斩。
但在战矛中蕴含的矛意也同样被次元斩的锋芒所磨灭。
“空间法术，竟然是空间魔法师。难怪能打开传送门，还敢对我进行围杀，还有谁，给我一起出来吧。”易天行深吸一口气，手中握着战矛，眼中一股战意疯狂的闪现。
嘴角边露出一抹冷笑道：“阻杀我，这是以为我好欺负么。对于任何挡在我面前的敌人，我都只有一个方法，那就是……..摧毁它！！”
一股冰冷的杀意瞬间迸发而出，席卷四方。
“易天行，你猎杀我西方修士，我们这是对你以牙还牙，同样施展阻杀。有什么不可以。”空间魔法师平静的说道。
“不错，你的比赛之路，到这里就可以终止了。”
一名身穿灰法袍，面孔显得枯槁，消瘦，看起来跟干尸一样的魔法师发出森冷的话音。手中的魔法杖都是由骨头制作而成，看起来就有些渗人。这是一名亡灵法师。
“你不死，我们心难安。”
又一名魔法师出现，在她身外，有清风环绕，散发出一股缥缈灵动的感觉。这是一名风系魔法师。
“这次为了阻杀你，我们可是连脸面都不要了，要是你不死，我们可就真的丢脸丢到爪哇国去了。以后也没有什么脸面出去见人。”
一名身穿重甲，手中握着一口白金盾牌，一手握着一口长刀，浑身肌肉都要爆炸的魁梧男子，跟铁塔一样从传送门中走了出来。
一出来，一股不动如山的浑厚气势，自然而然的呈现出来。
就那么站在那里，就能给人传递出一种强烈的安全感，仿佛有他在前面，那就是最安全的避风港，最坚韧的铜墙铁壁，没有任何危险能够降临在他们身上。这是一道不破的防御。
“一名重甲刀盾手，风系魔法师，亡灵法师，空间法师。一名强大的剑士。五尊强者，这是真的打算吃定我么。你们这是让我发飙，我发起飙来，连我自己都害怕。”
易天行深吸一口气，看着面前的五名强者，这已经是要置自身于死地的打算。
这样的敌人，何须留手，一个字，杀！！
“哦，快看，易天行之前的举动似乎已经触动到西方修士的敏感神经，现在已经被几名强大的西方修士出手围剿阻杀，三名魔法师，一名剑士，还有一名刀盾战士。防御，攻击，全部都已经做到极致。这次只怕易天行要有危险了，说不定会止步在这里。”
平台上，潇洒哥已经看到了这里。
这么精彩的画面，自然不能错过。
同样，在秘境中，无数大能的眼睛也都开始看向这里。
虽然现在的厮杀，战斗层次，对于他们来说，或许只是毫不起眼的一幕，但这里的却是未来永恒大陆上最杰出的一批天骄级人物。现在能看到他们之间的厮杀。更能看出未来的发展潜力强弱。
“不错，易天行的修为境界应该是神海境第六重左右的修为，虽然，修为并不代表真正的战力，但也代表着自身的一部分实力，至少他们面对的对手，一个个都有着不俗的战力。这个世界上，奇遇并不在少数，每天得到奇遇的更加不知道有多少。没有人是单独的幸运者。现在易天行所需要面对的，都是同阶中的强者。”
黑大帅大声说道。
言语中，似乎对易天行并没有多么的看好。
西方出手的阵容可以说是相当强大，在阵容上，已经没有多少的破绽。
“大帅，你不看好易天行，但我潇洒哥觉得易道友未必就没有胜算，毕竟，他的实力可不弱，而且，一直以来，都没有完全展现出自己的实力，他的箭术更加可怕，只要将距离拉开，未必就没有胜算。”
潇洒哥笑呵呵的说道。发表着属于自己的意见。
不管如何，他们的言语也干涉不了大战的结果。
“杀！！”
一道杀音瞬间出口，易天行身上已经快速被一副战甲所覆盖，在黑色战甲中，一股强烈的气势席卷而出。
嗥！！
金鹏听到，更是当场发出一声尖锐的厉啸声，在半空中朝着那名亡灵魔法师扑杀过去，带着易天行，当先发起攻击。
要战，那就由我来先出手。
易天行的性格，就是如此的直接了当，既然没有退路，那就杀出一条血路。
没有婉转的余地，那就直接动手。用手中的实力来说话。用手中的战矛来证道。来分出生死，强者就是道理，强者就是一切。
振臂，挥矛。
碎玉矛法。
一矛挥出，一股浓郁的碎玉矛意席卷而出，这种背水一战的局面中，让易天行对于碎玉矛法的感悟变得更加的深刻，更加的强烈。
面临绝境，无需惧怕，出矛，玉石俱焚。
宁为玉陨，不为瓦全！！
那股矛意，席卷而出，瞬间就让几名西方修士脸上浮现出一股强烈的凝重感觉，一种压抑不断的脑海中浮现。
“盾！！”
刀盾战士如铁塔般的身躯直接往前踏出，手中白金盾牌向身前一横，自身与盾牌之间，仿佛在刹那间，彻底的融为一体，自身就是盾牌，能阻挡任何风雨，任何的攻击，将一切的危险都可以抵御在外。他就好像是一块不破的铁壁，直接横档在战矛面前。
盾牌中，竟然同样传递出一股武道真意。
这是一种铁盾如山的武道真意，是盾意。如同磐石般的盾意。
砰！！
战矛以无与伦比的姿态重重的轰击在盾牌上，在盾牌上，闪烁着白金光芒，上面，似乎有无数神秘的魔法纹理在变幻。让盾牌上光芒璀璨。而且，强大的盾意，让盾牌的防御，更加强悍，这股意志，是不可摧毁的。难以破灭的。
“找死。”
西方修士同样反应过来。
同时爆发出强大的攻击。
只看到，风暴剑士手中的战剑豁然间刺出，瞬间爆发出无数道璀璨的剑光，一刹那间，覆盖易天行全身所有要害，锋芒可怕到极致。剑光中，同样蕴含着强大的剑意。
让每一道攻击，都狂暴，侵略如风。快如闪电。
速度极快，仿佛在一瞬间，已经刺出数百剑之多。
一道次元斩再次如匹练般朝着易天行脖子上劈斩下来。
这些，都发生在电光火石之间。
战矛已经轰击盾牌上。易天行感觉到，自己刺中的，好像是一座铜墙铁壁一样，十分坚韧，蕴含的力量相当可怕，从盾牌中传递出的力量，就算是一辆飞速行驶的汽车都能直接崩飞出去。
“好强大的力量，好惊人的盾意，不过，面对我，还是差了点，给我开。”
易天行心中暗自赞叹的同时，体内血肉中，黑鼎内气血疯狂的钻进每一寸血肉中，催动肉身力量，彻底灌注到战矛中，让战矛在一刹那间，蕴含着狂暴无比的力量。
“好强！怎么可能有如此强大的力量。他是人形凶兽吗。”
刀盾战士本身就是一名力量十分强大的战士，要不然，也不可能拿得起盾牌，成为一名真正的强者。但面对这一矛时，却感觉到战矛中的力量，仿佛山洪海啸般可怕，一股劲力后面，立即就有新的力量爆发出来，轰击在盾牌上，一次比一次更可怕。
就跟是潮汐一样。
席卷而来。一次两次，他还能抵挡住，但接连五六次下，根本没办法承受，本来站立在地面上的刀盾战士，一下就感觉到周身骨骼都在颤抖，血肉在颤抖，骨骼在断裂，五脏六腑都在出血。
噗！！
身体如遭雷噬般，猛地一颤，跟着一口逆血带着内脏碎片从口中喷出来。化为血雾，十分惨烈。
砰砰砰！！
面对一道道剑光剑意，次元斩，都没有半点退缩，身上真气灌注在战甲中，脚下却毫不犹豫的向前接连踏出三步，再次挥出战矛，依旧是那名刀盾手。
砰！！
这一矛挥出，刀盾战士本能的举起盾牌，挡在战矛前，当从战矛中爆发的恐怖力量，一下子就将盾牌强行从手中崩飞出去，一口鲜血再次喷出。
而后，只看到一道白玉般的流光闪过。喉咙中传出一阵剧痛。(未完待续。)
