永恒国度 第014章 坐忘经

发表于 2017-07-10 | 分类于 永恒国度

求收藏推荐，兄弟们给点力。
“紫嫣，开门，是我！！”
易天行松了一口气，当即就向里面张口呼喊道，同时按响门铃。
“是我哥回来了。”
在房中的赵紫嫣和唐梓彤都是手中拿着兵器，守在门口的位置，听到门铃与呼喊，连忙上前，从猫眼中立即就看到易天行的身影，脸上顿时露出一丝轻松的神色。
咔嚓！！
打开防盗门，易天行与陈雪柔快速进屋，同时，也以最快的速度将防盗门关上。
“哥，你没事吧，有没有受伤，我从阳台上看到外面很多怪物都在杀人，很多人都被敲碎脑袋，连尸体都被吃掉。”赵紫嫣将易天行身上的背包接了过去，眼睛在他身上来回打量，似乎在寻找有没有受伤的迹象。
“别找了，那些怪物还伤不了我，对了，这一位是陈雪柔。之前在外面碰到的，她暂时没有去处，就先到我们这里住一段时间。”
易天行笑着说道。
回到家中，本来绷紧的神经也不由开始放松下来。
家，就是避风港！！
“哦，雪柔，我是赵紫嫣，这位是唐梓彤。你叫我紫嫣，叫她梓彤就可以。你要没地方去，就先在我们这里留下来好了，正好一起有伴。”
赵紫嫣听到，也露出笑容，相互介绍起来。
“恩，谢谢你们，要不然我还真不知道该怎么办了。要我在外面，恐怕会被那些怪物给吃掉。”陈雪柔也露出感激之色。
三女都是同一级别的美女，美女之间总是容易亲近，尤其是在这种末日灾变的情景下。面对无数怪物从天而降，人与人之间总是更加容易相处，当然，这是米日刚开始而已，一旦持续下去，人心会以难以想象的速度发生改变。
黑暗，无情，绝望，死亡，将是其中的主题。
“哥，这些是什么，好漂亮的珠子。”赵紫嫣已经打开易天行的背包，顿时，一背包的物品呈现在眼前，那些愿力珠，每一枚都好像是最顶级的珠宝一样，让人忍不住心生喜爱。
“这些是愿力珠。杀怪物后得到的，每一头怪物死亡后，体内都会出现这样的愿力珠。根据了解，外面的大灾变与人类的执念，幻想，黑暗等等，诸多情绪愿力有着紧密的关联。那些怪物，都是幻想中存在的怪物，人类的虚无缥缈的幻想凝聚出的怪物。全部出现在现实世界。杀了他们，就能得到纯粹的愿力珠，这是宝贝。”
易天行深吸一口气，沉声说道。
对于外面的事情，也将自己所了解的一些情况说了出来。
现在这种时刻，能多一份了解，都能更快的适应这注定会与原先世界完全不同，彻底支离破碎的世界。
“还有这样的原因。哥，你怎么会知道的。”赵紫嫣听到，也是心神一震，不过，也好奇易天行是怎么知道的，她可不相信，仅仅出去一趟就能探查到这么多的消息。
“是陈雪柔，她手中有一件异宝，名为无字天书，能记录天地间的诸多大事，这次大灾变就在里面有记载。包括这次大灾变的来历，里面都有着详尽的记录。”
易天行看向陈雪柔直接说道。
“无字天书？还有这样的宝物。”
唐梓彤听到，也不由露出一丝惊讶之色。
这种宝物一听就不是普通的物品。
“这也是我运气好，之前天空中出现裂缝，里面有很多异宝从天而降，刚好这无字天书就直接落在我家，要不然，我也不会知道这些事情。无字天书能记录各种事件，能知道各种宝物的讯息。”陈雪柔谦逊的说道。
“我怎么就没有被异宝给砸中呢。”赵紫嫣听到有些郁闷的说道。
“好了，紫嫣，这种几率本身就很能得到异宝，那也是雪柔她自己的运气，羡慕不来的。”唐梓彤对这些倒是有些坦然。
毕竟，这种概率实在是小到可怜。
要不然，全世界的人谁都能被异宝砸到。
天上掉馅饼的好事可不是那么容易碰到的。
而且，就算是馅饼，也有可能被噎死！！
“哥，快看看这些都是什么，好多愿力珠。”赵紫嫣对这些亮晶晶的愿力珠也没有任何的抵抗力，眼睛都在发光，愿力珠比任何宝石都要来的更加珍贵神异。
“愿力珠是好东西，能对修炼有巨大好处，你们清点一下，找东西装起来，这些都是宝贝。以后你们要是开始修炼，这些就是最好的资源。里面蕴含的精纯香火愿力，能直接炼化成修为。远比正常修炼要快。”易天行颔首点点头。
愿力珠这在之前就已经查看无字天书，知道它的作用。那愿力珠，绝对是作弊器般的宝物。一枚白色愿力珠，若是全部炼化，那就相当于普通人苦修一年的修为。而要炼化愿力珠，则是看自身的功法如何，有快有慢，普通修士炼化一枚大概也就是十天半个月。
有天资卓绝，功法强悍的，有可能一天就能炼化一枚甚至是数枚。
但这绝对是提升修为实力的宝物。
不论多少都不会嫌多。
“恩，这些愿力珠最多，都快要把背包彻底装满了。我去拿箱子来装。”赵紫嫣立即兴奋的跑到卧室中，找出一只银白色的箱子。箱子是特质的，能防弹，以前易天行帮了一位土豪老板解决灵异事件，就特意送了一只箱子现金。
里面的现金不说，这箱子就不简单，防弹防火防水防震，特殊材料制成。还真不错，就留了下来。
现在正好派上用场。
一枚枚晶莹剔透的愿力珠不断被放进箱子内。同时也在默默清点着。
片刻间，所有的愿力珠已经全部清点完毕。
“一共是一千一百二十三枚白色愿力珠，还有三枚青色愿力珠。你竟然一下子杀了这么多怪物。”唐梓彤惊讶的看向易天行，刚刚也知道，一只怪物体内只会出现一枚愿力珠。
这里有一千多枚，也就以为着有一千多只怪物死亡。这是何等惊人的实力。
“很多并不是我杀的，能得到这么多愿力珠，完全是属于意外，机缘巧合下才发的一笔横财。”易天行笑着说道。
要没有阴兵借道的事情发生，要收集这么多愿力珠，简直就是不可能的事情。
“这光球中是什么。”
赵紫嫣好奇的看着那一枚枚光球，开口询问道。
“这光球中蕴含的是那些怪物体内的精华，一些由幻想愿力凝聚出的宝物，具体是什么，没有打开光球，我也不知道。不过，也有可能是一些炼器材料。”
易天行笑着说道。
对于这些光球也很期待。
“绿骨，绿色的骨头，看起来跟玉质的一样。这是从怪物身上得到的材料么。”唐梓彤拿起一枚光球，捏碎后，就出现一块碧绿色的骨头，看起来很漂亮。
“狼牙，骨材，可炼器，凡阶上品！！”
赵紫嫣拿起一枚光球捏碎后，里面出现一枚硕大如匕首般的狼牙，闪烁着寒光，锋利无比，能感受到其锋芒。在捏碎的一刹那，也明白这是什么材料。
似乎光球中本身就蕴含着一股讯息。
但在捏碎后，再拿起狼牙，就不会有这种讯息融入脑海。
这似乎是冥冥中的一种规则。
这种规则很玄妙。
而在无字天书上，也同样浮现出相同的记录，这无字天书，是随着阅历见识的不断增长而逐渐丰富内容的。显然，不愧是异宝。
“僵尸牙，骨材，药材。可炼器，可入药。黄阶下品。”
“狼皮，皮料，可炼器，可制衣，黄阶下品。”
一件件物品不断清点出来。
“哥，这枚光球中好像是功法。”
赵紫嫣突然开口说道。
只看到，在其手中，一枚光球已经捏碎。
出现在手中的是一本非金非玉，非绵非帛的奇特书册。上面还有淡淡的光晕闪过，古朴中带着神秘。
易天行伸手接过。
在那书册上，赫然浮现出几个充满神韵的古篆大字。
“坐忘经，神海境功法，修炼，可开辟神海。可修炼聚气，神泉，化海。完成神海境修行，内有观想图，先天一气观想图，于坐忘之间，观想先天一气，最终开辟神海，千人修行，千般坐忘。修行筑基极品功法。”
赵紫嫣在捏碎光球时，已经得到关于坐忘经的讯息。
“果然是修行功法，无字天书上也出现记载，这坐忘经是修行筑基的极品功法，同等层次中最上乘顶尖的功法，而且，修出的真气是最契合自身的，与自身的体质完美契合。火属性体质的，修出的就是火属性的坐忘经真气。还能随时进行转修其他功法，毫无隐患。”
陈雪柔也翻开无字天书，欣喜的说道。
“人有七情，心有六欲，坐而忘之，心如止水，神而明之，气自虚无来”
易天行翻开坐忘经，里面当先就是一段玄之又玄的总纲。字里行间，都蕴含着一种玄妙的神韵。让人深刻领会其中真谛。
而且，里面有观想图，是先天一气观想图，看起来只是一团先天之气，但却玄之又玄，不可名状。
