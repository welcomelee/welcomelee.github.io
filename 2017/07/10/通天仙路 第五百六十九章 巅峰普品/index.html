通天仙路 第五百六十九章 巅峰普品

发表于 2017-07-10 | 分类于 通天仙路

物品：撞角头盔
等阶：法器普品五阶
属性：锋锐+5，坚硬+5，耐久5
如意度：5
这就是欧阳明所锻造出的撞角头盔的真实属性，这是一件货真价实的普品巅峰法器。
只是，当欧阳明宣布了这件法器的属性之后，所有灵兽的目光中都有着一丝狐疑之色，这个人族锻造大师，不会是弄错了吧？
当然，肥羊和大黄对此却是有着极大的信心，大黄跟随欧阳明那么久，自然知道他的潜力和过人之处。而肥羊虽然没有与他们相处几日，但欧阳明给它打造的全套装备，却依旧是让它对其信心十足。
那不仅仅是法器，还是一件极其罕见的技能装备。
既然欧阳明大师连技能装备都能够打造，那么在锻造白板属性装备之时，稍微厉害一又算得了什么。
诸多灵兽们交换着眼神，如果欧阳明不是有着一件四阶普品法器装备打底，它们怕是早就起哄将他轰下去了。
那四不像的灵兽怔了半晌，接过了头盔瞅来瞅去，突地转身道：“林大师，您能帮我鉴定一下吗？”
欧阳明的嘴角微微抽搐了一下，心中暗恼，既然不信任我，又何必请我出手呢？不过，看了眼林飞沣，欧阳明还是决定，给他一个面子吧。
毕竟，这里是属于林飞沣的高台，自己虽说是被林飞沣邀请上来的，但好歹也受了他的指，不能喧宾夺主啊。
林飞沣微微头，将头盔拿在手中认真地检查着。
其实，此刻在他的心中，那种震撼和疑虑的感觉一儿也不比下面诸多灵兽要少分毫。
欧阳明可是在他的指下才明白锻造之术的那些最简单的道理，按照他的估算，就算欧阳明天赋惊人，但想要将这些知识顺利地运用到实践当中，也必须要足够的时间和精力才行。
多的不说，起码也要一年半载，或者是数月时间吧。
可是，欧阳明给他带来的惊喜实在是太大了，一转眼就锻造出了四阶装备，再一转眼，连巅峰普品法器都出来了。
这种一件装备上一个台阶的事情，突然发生在他的眼前，也难怪他难以置信。
不过，他比这些灵兽更强大的是，他有着辨识装备具体属性的能力。
接过了头盔，他前所未有的认真起来。
所有灵兽的目光都凝聚在他的身上，似乎在等待着他的判决，而林飞沣对于其余的目光恍若未觉，他专心致志地投入其中。
足足半刻钟，林飞沣终于抬起了头，在他的脸上闪过了一丝淡淡的疲惫之色。似乎适才的装备鉴定，消耗了他太多的力量。但实际上，这并不是他精力消耗太大所致，而是他的心太累了。
巅峰五阶普品法器，这果然是一件巅峰五阶普品法器。
可是，你一个刚刚学到这些知识的人，怎么能够连这种级别的法器也锻造出来了？这让我们情何以堪啊。
怀着根本就无法表述的心情，林飞沣长长的吸了一口气，正容对那只灵兽道：“巅峰五阶。”
“什么？”
“我说巅峰五阶，普品法器。”林飞沣没好气地道：“听明白了么？”
“啊，明白，明白了！”那灵兽一个转身，朝着欧阳明深施一礼，随后捧着头盔，跳下了高台，一溜烟的走掉了。
欧阳明微微摇头，这家伙虽然富有，但也太大惊小怪了。
这里既然是万兽尊者主持的会场，必然有着它颁布的规矩。若是有哪只灵兽鬼迷心窍在这儿动了妄念，它又岂能善罢甘休？
不过，这也是欧阳明身处旁观者的位置才能说得话，若是他身在其中，怕是也好不到哪里去。
“欧大师，请您也为我锻造一件这样的头盔吧！”另一道身影从台上响起，一只灵兽跳了上高台。
“不，我先来的，应该让我先来！”
“滚，后面排队去！”
一道道怒吼声相继响了起来，诸多灵兽们开始争执，场中的气氛逐渐变得火爆了起来。
欧阳明和林飞沣对望了一眼，面色变得凝重起来。
虽说在这里，他们不太可能被人针对。但是，如果这些灵兽真的因为愤怒而发狂起来，他们也会有着被波及的危险。
就在局面即将失控之时，突然一道高大的身影从半空中一跃而下。
“轰……”
一只体型巨大，足有五米之高的巨兽仿若巨石般的砸了下来，重重地落到了地面上。
欧阳明的眼眸一亮，这是一只拥有六只手臂的巨大金刚，它的身上散发着凶神恶煞般的气息，瞪着巨大的眼睛扫视了过去。
当这只金刚出现的那一刻，所有混乱的气息尽数消失，就算是有着灵兽即将失去理智，但是一见到那魁梧的身形，顿时恢复了神智，向着金刚露出了讨好的笑容。
六臂巨兽环目一圈，怒道：“你们吵吵嚷嚷的做什么，莫非不懂规矩了么？”
那些灵兽一个个开口，道：“大人息怒，我们只是想要请这位人族大师锻造装备，一时心急了，所以惊扰了大人。”
六臂巨兽狐疑地看着他们，道：“不就是锻造装备么，你们抢什么抢？！”
诸多灵兽唯唯诺诺，但心中却是腹诽不已。
这可不是一般的装备，而是巅峰五阶的普品法器。
灵兽，在万兽岭中并不稀奇。而事实上，灵兽之中也是等阶分明，初中高三级之外，还有顶级灵兽。
而此刻在交易会上想要求得一件合适装备的，大都是初级灵兽，以它们的实力，能够获得一件巅峰五阶的普品法器，就已经是这一阶中最好的待遇了。
这种可与而不可求之事，若是谁错过了，绝对会后悔终身的。
当然，这些话它们可不敢向六臂巨兽直言。
六臂巨兽划拉了一下巨大的手臂，虽然这手臂远不如多臂金刚那么庞大夸张，但也是气势万千，将这些低阶灵兽压制的动弹不得。
“你们，给我好好排队，若是再闹事，我将你们都扔出去吃了！”六臂巨兽瞪圆了眼睛恐吓着这些灵兽。
虽然这些灵兽适才表现得极为强势和暴躁，但是遇到更加强大的灵兽之时，它们却全部收敛了脾气，变得乖巧起来，让欧阳明心中诧异，这些灵兽的脾气也太好了吧。就算是明知不敌，也不可能吓成这样啊。
林飞沣轻笑一声，道：“欧兄，万兽尊者的执法者来了，我们可以不用担心了。”
欧阳明微微头，目光在这只六臂巨兽身上停留了一下，不知道这家伙是多臂金刚的弟弟呢，还是妹妹啊……
在这大家伙的安排下，一切都变得井井有条了，原先吵闹不休的灵兽们一个个变得要多老实就有多老实了。不过，当它们排成长队之后，都是微微有些发怔。
因为除了肥羊等袖手旁观的灵兽之外，其余灵兽都排在了欧阳明的面前，就连先前与林飞沣有约的那些灵兽也不例外。
欧阳明的脸色微变，道：“林兄……”
林飞沣摆了一下手，笑道：“欧兄，这很正常，它们辛辛苦苦收集了那么多矿石，当然想要更好的东西了。不过……”他顿了顿，朗声道：“我早就说过，想要让欧兄弟动手，需要双倍酬劳，你们自己估量着办吧。”
那一条龙排队的灵兽中顿时有好些变得犹豫起来。
灵兽如果在下界，自然是最顶尖层次的强者，但是在灵界，灵兽实在不算什么。它们之中拥有山头的灵兽还好一些，但那些流浪灵兽若是没有机遇的话，绝对是囊中羞涩。
林飞沣提了一倍的酬劳，对很多不宽裕的灵兽而言，已经是很难接受的了。
欧阳明轻咳一声，道：“各位，适才能够锻造出五阶普品，那也是一个意外，你们总不会以为，我每一次都能够达到这种水准吧？”他顿了顿，道：“就算有六份材料，但我只要锻造出二阶以上就会结束。你们……是否愿意继续冒险呢？”
诸多灵兽先是一怔，它们在心中盘算片刻，终于有些灵兽离开了这一排，到了林飞沣的面前。
它们的脸上都带着讨好的笑容，生怕林飞沣心生怨恨而拒绝。
然而，林飞沣只是微微摇头，便开始按照它们的要求开始锻造了。
其实，如果换了其他人如此做，林飞沣肯定是心生嫉恨。可是，欧阳明的锻造知识还是他亲自教导的，但一晃眼间，就已经超越了自己。
而且，他还注意到了一，那就是欧阳明所锻造的装备，如意度都是高达五。
这样的事实，让他连妒忌的力气都没有了。
与这种怪物比高低，那是需要何等的勇气才能去做的事情啊？
作为一个正常人，他的选择自然也是很正常的。
六臂巨兽瞪圆了眼睛看着这一幕，突地扯过了一只灵兽，问道：“什么五阶普品？”
那灵兽不敢隐瞒，道：“大人，这位人族大师刚刚锻造出一件巅峰五阶的普品法器，而且还是如意度高达五的装备！”
六臂巨兽眼眸一亮，它终于明白适才此地为何会如此热闹了。
对于这些最普通的初阶灵兽而言，为了这种级别的装备，确实是值得一战了。
