限制级巨星 第424章 骑虎难下

发表于 2017-09-08 | 分类于 限制级巨星

高小冬在华国的家乡度假的时候，欧洲转会市场依然在如火如荼的进行着。 更新最快
08年经济危机之后，世界整体经济尚未复苏，按常理，球员买卖这一“奢侈品交易”没有成交量大幅提升的空间。但由于皇马主席弗洛伦蒂诺强行挑起军备竞赛，彻底搅动了欧洲市场。
随后在不同的动力驱动下，欧洲转会市场波澜不断，从第一天就进入了爆点不断，每一天都有人在挥舞着支票簿，每一天都有球迷熟悉的名字在欧洲大陆上空飞行。
欧洲转会市场的第一个**是六月初皇马6500万欧元买来卡卡，这次转会成功，让弗洛伦蒂诺变得更加野心勃勃，卡卡税后900万欧元的年薪也让其他的巨星眼红心动。
6月27号，皇马又掀起更大的**，c罗正式加盟加盟。转会费为创纪录的9400万欧元，这是世界第一身价，年薪税后1300万欧元，这是世界第一年薪。
c罗的转会完成之后，手握大笔金钱的弗格森爵士再次以4500万报价波尔图，求购高小冬。
曼联的这个报价虽然无法和曼城相比，但相距已经不远，距离5000万欧元违约金500欧元的差距也完全是在提醒波尔图，曼联随时都可以再加上500欧元，绕过波尔图，直接找球员谈判。
在暗地里，曼联也第一次给高小冬开出了税后350万欧元的年薪。
虽然这个年薪无法和c罗、卡卡相比，但比起高小冬原来在波尔图的年薪却已经高出了四倍多。
国际米兰主席莫拉蒂看到曼联疯狂的出到4500万欧元的时候退缩了，他觉得用4500万欧元买高小冬这样的新星太冒险了，他直接打电话找到了穆里尼奥，表示不能再加价了，维持原来的3000万欧元加夸雷斯马的报价，来不来随便高小冬。
但穆里尼奥愤怒的表示，如果他不再加价，高小冬就是曼联的人了，在这个时候，国际米兰绝不能退缩，无论如何都必须和曼联争下去，他以帅位担保，高小冬绝对物有所值。
莫拉蒂这次没有无所则的退让，他阴沉着脸表示，他可以同意穆里尼奥和曼联打价格战，但如果高小冬和夸雷斯马一样水的话，后面他将收回穆里尼奥在转会上的权力，同时莫拉蒂还表示如果以这样的高价购买高小冬的话，他将卖掉球队的核心伊布拉希莫维奇。
一般教练很难接受这样的条件，但穆里尼奥不是一般的教练，他也明白，无论高小冬来或者不来，伊布恐怕都得走人，便欣然同意莫拉蒂的条件。
随后国际米兰宣布了两个重磅消息，一是国际米兰接受巴萨的4600万欧元加埃托奥的报价，同意伊布转会巴萨。二是直接以国际米兰史最高的5000万欧元转会费报价高小冬。
在私下里，穆里尼奥也通过门德斯向高小冬许下了400万欧元年薪的报价，这个价格比国际米兰队长萨内蒂还高了50万欧元。
这是高小冬的违约金，报价到了这个数字之后，国际米兰就可以不再理睬波尔图的意见，直接和高小冬联系，只要高小冬点头，国际米兰就可以直接把高小冬带到米兰城了。
曼联私下里和高小冬的约定是在相同条件下，高小冬同意加盟曼联，现在国际米兰出价高于曼联，那么高小冬转会国际米兰也不算是失信。
当然曼城的出价更高，给出的年薪更高，可是高小冬却并没有答应综合条件太差的曼城。
弗格森为了抢到高小冬做出了很多努力，不想在这个时候被人横刀夺爱，他也果断的把高小冬的转会费提到了5000万欧元，年薪也提高到了400万欧元。
溢价溢出到了这个地步，已经没有人再和曼联、国际米兰、曼城竞竞争，原来还想把水搅浑的，给隔壁抬抬价格的ac米兰觉得莫拉蒂疯了，吓得连价格都没敢报，就灰熘熘的直接撤退了。
曼联、国际米兰给出的转会费都已经达到了5000万欧元，高小冬的转会就和波尔图没有什么关系了，走还是不走，去哪里的决定权落到了高小冬的手上。
现在国际米兰和曼联给高小冬开出的年薪相同，按照高小冬和弗格森私下达成的协议，高小冬应该加盟曼联，除非国际米兰继续加价。
不过这个时候穆里尼奥也无法继续加价了，再加下去，国际米兰的工资结构就崩溃了，目前队长萨内蒂只有350万欧元的年薪，斯坦科维奇400万欧元，最好的边后卫麦孔才450万欧元，埃托奥虽然开出了1050万欧元，那是埃托奥降低了转会费，2000万欧元买埃托奥，这简直就跟白捡的差不多。
但穆里尼奥也是骑虎难下，都到了这个时候，跟莫拉蒂也保证过了，如果还是无法拿下高小冬，穆里尼奥恐怕要彻底成为意大利足坛的笑柄了。
上次的夸雷斯马就搞得穆里尼奥灰头灰脸，在意大利足坛抬不起头来，现在在和弗格森争夺高小冬上是不容有失。
在这个时候，穆里尼奥知道只靠价格上和曼联争的话，最后恐怕只能是两败俱伤，他果断的找到了经纪人门德斯，表示自己必须得到高小冬，但这个年薪不能再提高了，让门德斯想办法说服高小冬加盟国际米兰。
门德斯一直在隔岸观火，因为高小冬的身价飞涨对他是有利的，仅仅c罗转会皇马，他就拿到了800万欧元的佣金，如果高小冬再以5000万欧元的身价成交，他就要笑死了。
但穆里尼奥的存在对门德斯是很特殊的，穆里尼奥不仅是门德斯的摇钱树，而且有穆里尼奥在，门德斯旗下的球员可以轻松的宋到穆里尼奥的球队，这让门德斯可以轻松的吸引到很多高小冬一样的天才新星。
所以门德斯必须要保住穆里尼奥的金字招牌不倒，现在穆里尼奥尴尬处境门德斯也很清楚，他不想因为高小冬多拿百十万的年薪，就让穆里尼奥的国际米兰内乱，虽然高小冬也是门德斯非常看重的人，可是高小冬毕竟还是潜力新星，和c罗不能相比。
现在c罗转会基本结束，门德斯也轻松了下来，他便飞往华国，亲自说服高小冬转会国际米兰。(未完待续。。)
