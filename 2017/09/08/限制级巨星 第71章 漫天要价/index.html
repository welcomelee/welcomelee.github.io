限制级巨星 第71章 漫天要价

发表于 2017-09-08 | 分类于 限制级巨星

隔壁是肖乐和另外一个兴龙经纪公司的女员工的办公室，肖乐、高小冬他们进来后，女员工很知趣的离开了。?
双方落座之后，肖乐给四个人冲了一杯溶咖啡，坐在高小冬旁边，仔细听卡洛斯和佩雷拉讲话。
“高先生，我们能先听听你的想法吗？”卡洛斯用了很正式的称呼，而不是像在试训的时候，直接称呼高小冬的名字。
高小冬笑呵呵的道：“其实呢，我这人很朴实很低调，对什么要求都不高，我对加入波尔图只有两点想知道，一，我要踢什么比赛？二，给我多少钱？”
你朴实低调要求不高？那还开口就要钱，卡洛斯道：“先我得确定您的年龄，如果你17岁的话，那只能签青年队合同，没有工资，只有补助，而且无法在足协注册，无法踢职业联赛。”
高小冬猛然想起，现在元旦一过，已经是2oo7年了，他是1989年11月11日出生，虽然按照华国的习惯，都认为过了阴历的春节才是18岁，但按照国外的习惯，他已经是18周岁了。
高小冬拿出护照，递给卡洛斯的翻译道：“我已经18岁了，价真货实童叟无欺。”
卡洛斯接过护照仔细看了一下，又问了一下翻译，道：“既然你已经18岁了，那么可以签职业合同了，只要冬季转会窗口关闭之前给你在足协注册，你就可以打职业比赛了，不过，恕我直言，虽然你现在进步很快，但还是没有可能在葡打上比赛，所以，你只有两个选择，一个是打青年联赛，二是租借到葡丙、葡乙或者葡甲，如果有俱乐部要你的话。”
高小冬道：“那我能拿多少钱？”
卡洛斯道：“对于像你这样的年轻球员，我们一般会采取1+3的签约方式，第一年是临时合同，如果第一年展顺利，能够进入一线队，后面三年的正式合同就可以生效了。”
高小冬道：“总监大人，不要光绕弯子不说实话好不好，究竟多少钱，我就对数字敏感。”
尼玛，钱迷啊，卡洛斯无奈的道：“临时合同，我们给你最高等级的年薪，8千欧元，正式合同也给你年轻球员中最高等级的年薪，9万欧元。”
卡洛斯觉得自己已经给的不少了，临时合同就是最高等级，正式合同是第二等级，这是迫于主席的压力，他不想节外生枝，不然就给高小冬第三等级的年薪了，但没有想到高小冬听后，不仅没有他预想中那么高兴，反而撂下了脸。
尼玛，我在葡华队这样的业余队还有每月5oo欧的补助呢，去了豪门波尔图竟然跟业余队一样，你打要饭的啊，高小冬冷笑道：“卡洛斯总监，你是跟我开玩笑的吗？你知道我在葡华队的补助是多少吗？一个月5oo欧，你可以去问兴龙公司的陈总。”
卡洛斯苦笑道：“至于年薪，我觉得，像你这样年轻的球员，更多的应该考虑未来的展，而不是拿多少钱，一旦你有实力打葡，一年赚百万欧元都有可能。”
尼玛！觉得我胖就傻吗？拿展来忽悠我们年轻人，给自己省钱，展我有巨星系统，可以自己搞定，钱你们必须多给。高小冬连连摇头道：“不不，我更在意一年能赚多少钱。”
卡洛斯无奈，道：“那你说你希望拿到多少年薪吧？”
高小冬笑嘻嘻的道：“先，我不想要一年的临时合同，除非临时合同和跟正式合同一样的年薪。第二，波尔图这样的豪门，出手不能太寒酸了吧，至少也得2o万年薪吧。”
高小冬张口就把年薪加了两倍多，作为小商贩之子，胖子很懂得漫天要价摸地还钱的道理，宁可要跑了，不能要少了。
2o万欧元？你怎么不去抢？这已经是葡普通球队替补球员的年薪了，你一个18岁的毛孩子，要这个价格，尼玛，人家都说我们波尔图是黑店，这华国胖子比我们还黑啊。
这个价格，这个条件，卡洛斯实在无法答应，在波尔图黑店干青训总监这么多年，敢这样谈条件的年轻人，他还是第一次见，尤其是实力这么弱爆的，如果不是因为有主席达科斯塔的命令，卡洛斯肯定呸他一脸，然后拂袖而去，但是现在，他只能擦着汗道：“这个年薪我无法做决定，高先生，我最多只能给你到12万欧元的年薪，而临时合同是规定，还没有取消过的先例。”
佩雷拉再也忍不住了，道：“高小冬，你不知道行情不要随口胡扯，就算18岁的梅西、c。罗来波尔图，也不可能给他们开这个年薪的，俱乐部也不可能为你破例，取消临时合同。”
高小冬悠悠的道：“佩雷拉教练，都按照你们说的签合同，那还谈什么。”
佩雷拉还想说什么，卡洛斯怕激怒高小冬，连忙拉了他一把，道：“既然高先生坚持这个年薪，那等我们回去征求科斯塔主席的意见再给你答复吧。”
高小冬笑嘻嘻的道：“好的好的，我在金钱的问题上一直很有耐心。”
卡洛斯、佩雷拉郁闷的过来和门德斯会合，一起离开了葡华队的训练基地，临走的时候，门德斯笑容满面的给看高小冬一张精巧的银色名片，道：“小伙子有前途，以后有什么困难，给我打电话。”
高小冬接过名片，入手沉甸甸的，竟然不是纸质的，等门德斯离开之后，高小冬拿着名片问陈强，“这是银的吗？”
陈强接过来掂了掂，道：“是的，听说门德斯有四种名片，金银铜和镶钻的，这是银片。”
高小冬开心的笑道：“见面就送人几十欧，真牛逼。”
陈强心事重重的道：“这可不是什么人都送的，，今天给了你一张银质名片，是很看重你。”
高小冬笑道：“他是谁，看重我干什么。”
陈强嘿嘿了两声，正想怎么回答，这个时候博阿维斯塔的球探西蒙跟俱乐部的主席和青训总监已经通过了电话，走过来了。
陈强便对高小冬道：“先跟他们谈着，不要轻易许诺，回头我们再商量。”
高小冬笑道：“我知道，不货比三家，我不会答应任何一家的。”
陈强道：“你明白就好，不过也不要说绝了。”
“好的。”
高小冬过去迎向西蒙，道：“西蒙球探，我们来谈谈吧。”
“好的。”
西蒙球探刚刚从主席那里得到了允诺，可以给高小冬一个大合同，进了办公室之后，他马上兴冲冲的道：“高先生，我给你申请下了一份大合同，4+3，四个月的临时合同，3年的职业合同，临时合同，一个月1千欧元，职业合同年薪1o万欧元，这样的合同，我保证，在葡萄牙是最顶级的青年合同。”
西蒙说完，充满期待的看着高小冬，他以为高小冬会欣喜若狂，一口答应下来，毕竟高小冬只是一个免费踢业余比赛的华国少年。
然而高小冬却不仅不高兴，还有点讥诮的道：“西蒙球探，你知道波尔图的卡洛斯总监给我多少年薪吗？”
西蒙心一沉，摇摇头，道：“不知道。”
高小冬竖起两根手指，道：“年薪15万欧元，还不需要签临时合同。”
“这不可能！”西蒙的嘴巴张的老大，可以塞进一个鸡蛋了，“梯队的球员要是都开这样的年薪，葡球队根本负担不起。”
“不信你可以去问问卡洛斯。”高小冬笑笑，很无辜的道：“这个数字很高吗？我还以为是正常数字。”
尼玛，波尔图这是恶性竞争啊，这样的球员开出这样的年薪！西蒙也流汗了，道：“这个年薪太高了，我无法做主，高先生，这样吧，我回去再和主席、总监商量一下，然后再给你答复。”
高小冬笑眯眯的道：“可以可以，我不急，关系职业生涯的事，不能急啊。”
“好的，再见。”西蒙满腹疑云又满腹牢骚的离开了葡华队的训练基地。
看看人都走光了，陈强问高小冬，两家俱乐部给你开了什么样的合同。
高小冬把跟两家谈判的过程跟陈强详细的说了一遍，陈强听完之后，眼睛睁得滚圆，道：“天啊！这么多，你竟然还没答应。”
高小冬笑道：“我也不知道行情，正想问你呢。”
陈强苦笑道：“你知道希望之星的队员们工资是多少吗？队长张弛的在里斯本竞技青年队的年薪也是1+3，临时合同只有5ooo欧，正式合同也只有七万欧，本菲卡的杨帆和余德宝和张弛差不多，而其他去葡乙和葡丙的一年只有五六千欧，听清，不是一个月，是一年，虽然有出场费和进球奖，但是一年也不会过8ooo欧的。”
高小冬挠挠头，道：“这么说2o万年薪要高了。”
陈强喃喃的道：“何止是高了，太高了。”
高小冬笑眯眯的道：“高了就高了吧，漫天要价摸地还钱嘛。”
陈强无语，这小胖子，简直是妖孽啊。
