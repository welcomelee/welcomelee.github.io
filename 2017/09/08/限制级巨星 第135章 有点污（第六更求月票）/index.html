发表于 2017-09-08 | 分类于 限制级巨星

高小冬在里斯本波尔特拉国际机场候机厅等飞机的时候才知道希望之星的大半个球队都坐这个航班。
希望之星的球员和李胜坐在前排、陈强、高小冬、杨帆、余德宝坐在后排。
“陈哥真是的，提前也没给我说一声，你说我这被开除的人，和这些希望之星坐在一起，这压力得多大。”
高小冬对陈强说的，不过声音大的整个大厅都能听见。
这厮，和李胜有恩怨，得空就让李胜难受难受。
其实李胜看着高小冬比高小冬看他还不爽，但他又拿高小冬没有办法，这个胖子现在在葡萄牙混得风生水起，门德斯、科斯塔、卡鲁斯这些大人物都和他关系不错，别说高小冬只是指桑骂槐的刺激他，就算当面骂，李胜也只能唾面自干。
杨帆听高小冬这样说，怒道：“尼玛，我们都打葡乙，葡甲替补，你特么冠军队主力，明年就打葡，竟然说这么无耻的话，跟你一起坐飞机得小心，特么万一你装逼太过，天打雷劈的，我们也跟着遭殃。”
“陈……那个陈总……可能是想让我们互相有个照应。”余德宝本想喊陈叔，可是高小冬喊了陈哥，他这个陈叔就叫不出口了，便改称陈总。
“照应个屁，不给我使绊子就不错了，你没看到，一个个看我横鼻子竖眼的。”
其实希望之星的球员对这阴损的胖子是又羡慕嫉妒又畏惧忌惮，高小冬看过去的时候，都没有人敢跟他对视，这厮竟然还说别人看他横鼻子竖眼，也是没谁了。
余德宝不想队友被高小冬讽来刺去的，道：“小冬这次夺冠，奖金拿了不少吧。”
高小冬笑嘻嘻的道：“大宝，我特么不想装逼，你这不是硬逼着我装逼嘛。”
高小冬说话声音太大，吵到了不远处一位戴着墨镜玩手机的美女，她有些厌恶的向高小冬这边瞥了一眼。
陈强看有人往这边看，道：“小声点胖子，注意影响。”
“胖子。”杨帆道：“有美女看你呢。”
高小冬望向那个穿着红色裙子的美女，椭圆的脸型，白皙的皮肤，非常饱满的胸部，还有一双大长腿，除非墨镜下面没有眼珠，不然是美女无疑。
“少见多怪，你胖哥哪天没有百儿八十个美女欣赏。”高小冬洋洋得意的道。
美女看到了高小冬在看她，也听到了高小冬的话，她隔着墨镜瞪了高小冬一眼，掏出一副白色的耳机戴上，开始闭目听mp3。
杨帆哈哈大笑，“每天有一百多美女欣赏的胖哥被人鄙视了。”
高小冬一点也不觉得脸红，恬不知耻的低声道：“浅薄，说不定美女正在意淫我这壮男呢。”
卧槽！
杨帆和余德宝恶心的差点把早餐都吐了出来。
俗话说无巧不成书，实际上上现实有时候比书上写的还要精彩。
上了飞机之后，高小冬惊讶的现，他竟然和那个戴着墨镜的美女座位是挨着的。
墨镜美女身高一米七五以上，还穿着高跟鞋，如果高小冬不是很壮的话，站在她的身边就跟一个孩子似的。
坐在高小冬前面的杨帆看了忍不住大笑，“缘分啊，胖子。”
高小冬笑眯眯的道：“那咱们换换。”
杨帆一脸鄙视道：“屁股都没动，一看就没诚意。”
美女修眉微蹙的坐下了，她有点惊讶，但更多的是不爽。
高小冬和杨帆其实很久都没见过了，现在坐到一起，自然扯个没完，美女刚开始戴着耳机听音乐，还觉不到什么，后来mp3没电了，就受不了了。
“我说两个小弟弟，你们少说两句，没人把你们当哑巴。”
美女是华国人，操着一口堪比播音员的普通话，声音很好听，但是内容却杀伤力十足。
如果美女客气一点，高小冬也就不说了，毕竟打扰人家了，但是美女这么不客气，高小冬当然也不是什么善男信女，他笑眯眯的道：“小弟弟？大姐，不小了，18了。”
杨帆这厮也是个贱人，他故作惊讶状，“18？卧槽，欧美标准尺寸啊。”
一般的女人，听到这样的话，估计不是羞怒交加，就是低头不敢说话了，但是这位美女也不是普通人物，她微微一笑，道：“呵呵，连姐姐也敢调戏，胆子不小嘛。”
高小冬也笑眯眯的道：“都不小，用过的人都说好。”
“耳听为虚，眼见为实，中国男人就喜欢吹牛，有本事亮出家伙来看看嘛。”
墨镜美女摘下墨镜，露出一双丹凤眼，果然美得冒泡，高小冬和杨帆都快看直眼了。
美貌惊人，说话更是惊死人。
高小冬和杨帆毕竟还是少男，哪见过这么污的女孩，都傻眼了。
高小冬认输，“厉害了我的姐，我服了。”
墨镜美女笑笑，道：“你们不是很流氓嘛。”
高小冬笑道：“谢谢美女姐姐夸奖，咱们彼此彼此。”
墨镜美女笑道：“你叫什么名字？”
高小冬道：“美女姐姐，小弟姓高名小冬，鲁城人士，美女姐姐芳名呢。”
这厮是打蛇随棍上啊，杨帆看高小冬卖弄着不知道从哪个电视剧里学来的对话，差点吐了，尼玛，就这高考抄袭才考了四百分的水平，竟然还拽文。
但是美女喜欢啊，美女笑道：“我叫秦宁，听你们刚刚聊天，你在葡萄牙踢球？”
偷听我们聊天啊，高小冬道：“嗯，姐姐做什么工作。”
秦宁笑道：“你猜猜。”
搞笑的略一沉吟，道：“美女姐姐面对杨帆那流氓乌七八糟的调戏都能面不改色，从容应对，这肯定不是一般的女性能够做到的，而且还这么漂亮，又带着墨镜，所以，姐姐不是模特就是歌星或者影星。”
杨帆听了，气得鼻子都歪了，尼玛，明明是你耍流氓，竟然全推到我身上，真是从未见过如此厚颜无耻之人。
这小孩子不简单啊，秦宁脸上微微露出一丝惊讶之色，“呵呵，猜的不错，我是毛豆。”
高小冬叹息道：“羡慕当模特的，每天往那儿站一会，就能进日进斗金。”
秦宁道：“那你们不更值得羡慕，每星期踢9o分钟比赛就可以日进斗金。”
高小冬笑了，道：“姐姐好机智，确实，每一个行业都不容易，还是做一个富二代好啊。”
秦宁道：“现在投胎也晚了，还是努力去做个富一代吧，别让你的儿子也像你这样感叹。”
高小冬笑道：“我一个人造不出儿子啊，姐姐身边美女多，可要帮我介绍个美女给我认识。”
“没有问题，只要你养的起。”
三人又闲聊了一会，秦宁道：“困了，我眯一会，别吵我。”
高小冬笑笑点点头，没说话。
秦宁向高小冬挑了挑大拇指，把墨镜往脸上一盖，仰在座位上睡了。
一路无话，高小冬不过打了个瞌睡，飞机已经抵达了巴黎国际机场，等高小冬被杨帆叫醒的时候秦宁已经下了飞机，高小冬非常遗憾没有能够向美女要手机qq号。
转机要两三个小时，陈强带着高小冬、杨帆、余德宝下机后在免税店买了不少回家送亲友的礼品，接着转机飞往华国都燕京国际机场。(未完待续。)
