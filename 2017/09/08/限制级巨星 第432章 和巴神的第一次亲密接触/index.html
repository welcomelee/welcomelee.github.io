限制级巨星 第432章 和巴神的第一次亲密接触

发表于 2017-09-08 | 分类于 限制级巨星

为了给新队友和新教练留下一个良好的印象，签约后的第二天，高小冬比训练时间提前了一个小时打车来到了皮内蒂纳训练基地。
训练基地的改造还没有完工，工人们正在施工，不过主训练场已经可以使用。
这块训练营是莫拉蒂的父亲安杰洛·莫拉蒂在巫师埃雷拉的要求和指挥下于1960年代修建的，不仅破旧，而且不适应现代足球的训练要求。
据说球员宿舍很糟糕，冬冷夏热，著名球星维埃拉曾经热得住不下去，半夜里偷偷逃出基地，跑回家休息去了。
曼奇尼在训练前也从不让球员住在皮内蒂纳基地里，都是到外面去找星级酒店住。
训练基地改造是穆里尼奥要求的，注重细节的狂人无法接受在这样的训练基地长期训练。
莫拉蒂对穆里尼奥还是非常支持的，不仅出巨资改造基地，而且准备购买皮内蒂纳之外合计超过1万平方米地皮全部收为训练营。
高小冬觉得自己来的够早了，没想到他来到的时候，穆里尼奥早就到了，正在和同样早早来到的球员们一起说话。
看着高小冬来到，穆里尼奥带着一个金发年轻人走了过来，指着年轻人向高小冬道：“小冬高，这是俱乐部给你请的翻译西奥，西奥精通华语和意大利语，有什么事情你可以向他寻求帮助。”
西奥用流利的汉语道：“你好，高小冬先生，我不仅是你在球队的翻译，还是您的意大利语老师，我愿意随时为您效劳。”
我擦，这华语说的，比我都溜，高小冬不禁大为佩服，道：“别客气，以后你就是我的喉舌和耳朵了，如果你能让我满意，我会给你发奖金的，但如果你出什么幺蛾子，我也不会客气的。”
即便西奥在华国留过学，汉语精通，也不知道幺蛾子是什么，但他大约也猜出了高小冬的意思，微笑道：“张先生放心，我是有职业操守和做人原则的。”
看到高小冬和翻译认识了，穆里尼奥道：“你跟我过来，我给你介绍一下新队友。”
球队虽然在米兰集合，但并不在米兰集训，国际米兰的夏季训练的地点放在了美国的洛杉矶，在米兰呆三天之后就飞往洛杉矶，南美的球员大多数都直接在洛杉矶和球队会合。
穆里尼奥虽然亲自出马，并且隆重的把高小冬介绍给了队员们，但国际米兰的球员对待高小冬仍然很冷淡，只是象征性的拍拍手，显然，国际米兰的球员对第二年轻的高小冬拿到全队的第二高薪是满肚子怨言的。
唯一觉得无所谓的只有拿着第一高薪的埃托奥。
穆里尼奥介绍完之后，大家各自活动，意大利天才巴洛特利走到了高小冬身边，围着高小冬转了两圈，“听说你很吊，昨天有三万球迷来看你？
高小冬笑嘻嘻的道：“不吊不吊，其实球迷都是来看我笑话的。”
巴洛特利道：“有道理，你长得不行，没有我高，也没有我帅，球迷肯定是来看你笑话的。”
高小冬笑嘻嘻的道：“是的，也没有你黑。”
巴洛特利接着道：“你没有我高，没有我帅，为什么我年薪10万，你年薪600万。”
周围的球员们都幸灾乐祸的看着巴洛特利找高小冬的麻烦，却没有一个人阻止，因为本来就是他们怂恿的巴洛特利。
这就是一**青年啊，对这样的人根本不能讲道理，威胁利诱也都不顶用，高小冬眼珠一转，决定用对付邻居家小崽子铁蛋的方式来跟巴洛特利交流，他笑眯眯的向巴洛特利道：“600万和1000万哪个大？”
巴洛特利咧嘴一笑，“这个还想难住我，当然是1000万大。”
高小冬道：“10万是你，600万是我，1000万是埃托奥，都是黑人兄弟，没你高没你帅，技术还不如你好，年龄还这么大了，凭什么埃托奥是你的十倍，你说是不是，马里奥。”
高小冬亲亲热热的揽住了巴洛特利的肩膀，一脸为巴洛特利不忿的样子。
巴洛特利脑子只有一根筋，高小冬这样一说，巴洛特利马上忘掉了高小冬的高薪，转身去找埃托奥。
高小冬的高薪国际米兰的球员都不服，但埃托奥的高薪却没有人有意见，埃托奥是两次欧冠冠军的主力前锋，又是作为伊布转会的零头来的国际米兰，一千万的年薪真的不算多。
马特拉齐看到巴洛特利不知道怎么回事，没找高小冬的麻烦，反而突然奔着埃托奥去了，他赶紧叫住巴洛特利，“马里奥过来，跟我一起去拿器材。”
巴洛特利不理继续走向埃托奥，马特拉齐过来拉住了巴洛特利，“你想干什么？”
巴洛特利道：“我问问埃托奥哪点比我强，年薪这么高。”
马特拉齐沉着脸道：”胡闹，教练让你去拿器材。”
高小冬笑眯眯的走过来，向巴洛特利和马特拉齐道：“我跟你们一起去拿器材吧。”
马特拉齐看到高小冬不动声色的把巴洛特利推向了埃托奥，现在又无事献殷勤，觉得这小子不是个好东西，不过他没有拒绝，也没说话，转身向器材室走去。
高小冬并不在意马特拉齐的态度，他只是要造成一种和意大利本土帮亲近的姿态就够了。
器材拿出来，球员们到齐，训练的时间也到了。
球员们刚刚度过一个假期，穆里尼奥并没有马上进行高强度的练习，上午主要是恢复训练，下午进行了传球和射门训练。
在训练中，高小冬展示出来的球技让穆里尼奥为首的教练组和国际米兰的球员们都非常吃惊。
这倒不是说国际米兰的球员觉得高小冬水平有多高，而是他们此前对高小冬的了解和认识都是建立在高小冬升入职业a级之前的比赛上的，高小冬升入a级之后，只打了和巴萨的半场比赛，这半场比赛，球权还都在巴萨的脚下，高小冬根本就没碰几下球，穆里尼奥和国际米兰的球员都觉得高小冬的技术很一般，现在近距离一看，高小冬每一个球处理的都非常合理，非常轻松，球性非常熟稔，根本不像一个不到0岁的毛头小子，就是一个老球痞啊。
穆里尼奥百思不得其解，距离欧冠决赛不到两个月，而且还是度假当中，高小冬的球技怎么突然又提高了一大截，这难道真是门德斯说的东方顿悟式提高，不然难以解释啊。
国际米兰的球员也收起了几分轻视之心，觉得这小子虽然配不上5000万的转会费和600万的年薪，但还是有几分斤两的。
