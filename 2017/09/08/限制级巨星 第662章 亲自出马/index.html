发表于 2017-09-08 | 分类于 限制级巨星

无论在淘汰巴萨和晋级决赛上，穆里尼奥和高小冬谁的功劳最大，总之，穆里尼奥和高小冬的名气更大了，粉丝更多了。
很多人都认为如果国际米兰能够在伯纳乌击败曼联，赢得欧冠冠军，穆里尼奥将成为当今第一名帅，而高小冬也可能成为第一个获得金球奖的亚洲球员。
由于高小冬在半决赛中展示出了超强的能力，极其全面的技术和巨大的潜力，不等赛季结束，争夺高小冬的转会战斗就提前打响了。
国际米兰的联赛冠军已经拿到手，欧冠决赛要在三周之后进行，意大利杯决赛在一周后进行，对国际米兰来说，是可以放松一下了，比赛的第二天，国际米兰并没有离开巴塞罗那，穆里尼奥给球员们放了一天假，让他们在这座艺术之都玩一天，第三天再回意大利准备和罗马的意大利杯决赛。
埃托奥很感激高小冬给他的助攻，正准备带高小冬在巴塞罗那玩玩，高小冬接到了他的在国内的经纪人陈强的电话。
和陈强通完电话之后，高小冬脸上露出了一丝很奇怪的表情，他跟埃托奥说自己华国的朋友来到了巴塞罗那，就打的离开了酒店。
在一家隐秘的私人会所，高小冬见到了陈强和真正想见他的人——巴萨主席拉布尔塔。
“小冬，这是巴萨主席拉布尔塔。”
陈强向高小冬介绍道。
拉布尔塔见到高小冬赶紧站起来过来跟高小冬握手。
“主席好。”
昨天在比赛前后和巴萨闹了一系列不愉快，高小冬对拉布尔塔也有点反感，并没有给他什么好脸色。
拉布尔塔并不在意，笑道：“高，这次我们巴萨可是栽在你的手上了，没有你，也许进决赛的就是我们了。”
高小冬淡淡的道：“主席西过奖了，这不是我和巴萨的比赛，这是两个球队的较量。”
拉布尔塔竖起大拇指，道：“没想到看起来很傲慢自大的小冬高私下里竟然是这么谦虚的一个人。”
高小冬道：“谦虚是我们华国人的美德。”
“高，你知道我今天为什么会出现在这里吗？”拉布尔塔跟高小冬寒暄了两句，终于说到了正题上。
高小冬原来就听门德斯说过，巴萨想要买他，不过他故作糊涂，道：“听陈哥说，主席想认识认识我。”
拉布尔塔打了个哈哈，道：“是是，不过呢，我还有个想法，不知道你有兴趣听听吗？”
高小冬无可无不可的道：“既然我已经来了，主席有什么就直说吧。”
拉布尔塔道：“不知道你对巴萨了解多少，我觉得还是先从巴萨的历史讲起吧……”
高小冬打断了拉布尔塔的话，微带讽刺的道：“主席，巴萨的光辉历史就不用讲了，现在我多少也了解了一点，我觉得你有什么想法还是直接说好了。”
拉布尔塔知道高小冬说的是巴萨赛前赛后一些不择手段的行为，有些尴尬，道：“高，你可能误会了，哪家球队都有些极端球迷，就像国际米兰极端球迷组织也很多，其实和球队的传统是没有什么关系的。”
高小冬笑道：“主席不用解释，我也并不在乎，竞技足球就是胜利的足球，一切行为都是为了胜利，多年之后，人们记住的只有谁是冠军，细节早就特么的淹没到历史长河里了。”
拉布尔塔皱了一下眉头，心道，这小子不愧是穆里尼奥的弟子，把功利主义足球学得很扎实。
其实高小冬年纪轻轻，自然没有这个见识，这些都是穆里尼奥经常说的话。
拉布尔塔道：“巴萨的传统是攻势足球和传控足球，无论面对任何对手，巴萨从不防守反击，所以我们可以花重金买锋线上的巨星，极少花重金买防守球员，你上个赛季在波尔图的后腰前腰位置都表现不错，进球也多，但我们并不却前腰和后腰，所以没有出手跟曼联和国际米兰抢夺你。这个你在国际米兰也展示了极强的进球能力，巴萨的球探和教练组经过研究，认为你完全可以胜任巴萨的边锋和中锋，希望你能够来到巴萨和我们共创霸业。”
拉布尔塔解释了上个赛季没有买高小冬的原因，然后表达了希望高小冬加入巴萨的想法。
高小冬没有打断拉布尔塔的话，仔细的听完之后，他严肃的道：“拉布尔塔主席，如果要讨论转会的事情，我觉得你找错人了，在欧洲，门德斯先生才是我经纪人，您该找他谈谈，陈哥只是我在亚洲的经纪人。”
拉布尔塔哼了一声，道：“门德斯那里我也找过两次，一次打电话，一次亲自和门德斯先生见了面，但门德斯先生满口答应会好好跟你商量一下，但我一直等到半决赛打完，也没见门德斯先生跟我联系，我开始不知道是怎么回事，后来才知道，原来穆里尼奥先生下个赛季要去皇马执教，他当然不想让高转会我们巴萨，所以我才通过陈先生直接联系了你。”
陈强道：“小冬，是这样的，拉布尔塔主席希望直接听听你的想法，而不是门德斯传递给他的消息，在穆里尼奥准备下个赛季执教皇马的情况下，拉布尔塔主席不相信门德斯的话。”
高小冬眉头紧皱，道：“拉布尔塔主席，你怎么确定穆里尼奥教练下个赛季会执教皇马吧？穆帅的合同还有两年呢。”
拉布尔塔冷笑一声，道：“如果是冬歇期之前，我不敢说，但现在，十有七八，第一，欧冠击败我们之后，估计曼联也挡不住国际米兰夺冠，意大利杯夺冠的希望也很大，穆里尼奥本赛季在意甲可以说非常成功了，很难再有更大突破，穆里尼奥是什么人，会不趁着巅峰离开吗？第二，穆里尼奥的球队对巴萨的战绩很好，这是非常吸引皇马的地方，皇马必定竭尽全力买穆里尼奥；第三，穆里尼奥和国际米兰的条约里有买断自己合同的条款，所以穆里尼奥离开国际米兰一点法律障碍都没有……”
拉布尔塔说前面两条的时候，高小冬并不在意，这些大路边的道理，各大媒体都分析了不知道多少遍，但第三条说穆里尼奥能买断自己的合同，高小冬倒是没有听说过，这让高小冬小吃了一惊。
“……第四，在昨天比赛之后，穆里尼奥突然说他永远不会执教巴萨，这分明是向皇马示好效忠，如果我还不能确定他下个赛季执教皇马，那么我就白当这么多年巴萨主席了。”
拉布尔塔非常自信的侃侃而谈道。
不得不说，拉布尔塔分析的很有道理，但高小冬沉吟了一会，道：“对不起，拉布尔塔主席，我无意转会巴萨。”
拉布尔塔道：“高，我相信穆里尼奥一定希望把你带到皇马，但放心，无论皇马给你什么待遇，巴萨都会比皇马高，而且我保证会给你绝对主力的位置。”
高小冬笑了笑，道：“真的吗？主席，如果皇马给我的年薪超过了梅西呢？”
“这……”拉布尔塔张口结舌说不出话来。
