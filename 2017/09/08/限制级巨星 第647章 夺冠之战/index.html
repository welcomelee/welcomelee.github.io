限制级巨星 第647章 夺冠之战

发表于 2017-09-08 | 分类于 限制级巨星

第二场欧冠半决赛，曼联同样打了里昂一个4:0.
这样以来，欧冠决赛的形势就非常明朗，除非出现惊天大冷门，基本上出现在伯纳乌的就是国际米兰和曼联了。
赛后博彩公司已经不愿意再为里昂和巴萨的夺冠开赔率，因为在欧冠历史上，还从来没有任何一支球队落后4球还能逆转的。
即便如此，国际米兰依然不敢大意，在周末和保级球队亚特兰大的第35轮联赛中，哪怕有希望提前三轮夺冠，注重实利的穆里尼奥还是让主力们休息，排出了一个替补为主的阵容。
高小冬是国际米兰攻防两端的核心人物，穆里尼奥本来是希望高小冬放弃这场比赛的，但却被高小冬拒绝了。
穆里尼奥肯定是很不爽的，他觉得国际米兰领先7分，早晚都会夺冠，在现场捧不捧杯的无所谓，一场无关紧要的比赛，高小冬完全没有必要上场，万一影响到了客场和巴萨的比赛，那就有得不偿失了。
不过高小冬转会国际米兰之前和穆里尼奥有过协议，他要整个赛季的首发，除非他自己放弃，穆里尼奥也答应了，所以穆里尼奥只能心里不爽，却没有办法强行把高小冬放在大名单之外。
穆里尼奥以为高小冬是虚荣心太强，想和米利托、迪纳塔莱抢最佳射手，而高小冬实际上却是为了提前夺冠晋升职业s级。
这一轮第二名罗马遭遇强敌，受到第四名桑普多利亚的挑战，罗马想赢桑普多利亚并不容易，只要罗马不能赢球，而国际米兰击败亚特兰大，国际米兰就可以提前三轮夺冠。
高小冬现在就缺少一个联赛冠军，只要有了联赛冠军，高小冬就能够升上职业s级，这是介于普通球员和球星之间的级别，一旦晋升职业s级，高小冬的实力就会得到更大的提升，对付巴萨就更有把握，所以，就算消耗一点体能，对高小冬来说也是绝对值得的。
目前亚特兰大排名倒数第三位，仅仅高于铁定降级的利沃诺和锡耶纳，比倒数第四名博洛尼亚少5分，还是有一线保级希望的。
虽然亚特兰大是一个鱼腩球队，一个赛季换了三任教练都没有改变颓势，但从历史战绩来看，和国际米兰球衣差不多的亚特兰大还真的不怕国际米兰，上个赛季国际米兰客场1:3输给了亚特兰大，主场4:3险胜，在垂死挣扎之际，说不定能趁着国际米兰分心爆个冷门。
像对阵亚特兰大这样的比赛，如果在上个赛季，看台上估计连一半球迷都坐不到，但本赛季国际米兰战绩辉煌，一扫数十年之颓势，大有拿下意甲第一个三冠王的架势，而且这场比赛国际米兰有可能提前三轮夺冠，为了能够目睹国际米兰现场捧杯，梅阿查球场的看台爆满，而且意大利足球联盟的官员们也带着奖杯来到了现场，一旦国际米兰提前夺冠，赛后就将进行颁奖仪式。
虽然穆里尼奥不愿意派出主力阵容，但并不是不想赢球，穆里尼奥也希望在和巴萨决战之前把后顾之忧全部解除，只不过他不想在亚特兰大花费太大的力气罢了。
比赛开始之前，穆里尼奥还是进行了认真的战术安排，强调不要给对手希望，开头就要击溃亚特兰大的信心，像这种疲软了一个赛季的鱼腩球队，只要没有了信心，基本上也就任人宰割了。
如今穆里尼奥在国际米兰队内威望空前，他的话在国际米兰就跟圣旨一般，哪怕是替补球员，对穆里尼奥的战术安排也会不折不扣的执行，战斗力远远不是亚特兰大这种球队可比的。
开场之后，在全场六万多球迷的加油助威声中，一直未能获得出场机会的国际米兰替补们在高小冬的带领下把亚特兰大压在半场狂殴，比赛基本上变成了半场攻防。
然而足球是圆的，这句话一点都不假，就在国际米兰的球迷都翘着脚丫嗑着瓜子等着欣赏国际米兰虐杀亚特兰大的时候，亚特兰大竟然利用一个反击率先攻破了国际米兰的大门。
比赛第十分钟，亚特兰大首次射门，曼弗雷迪尼左路长距离直塞，蒂里博基反越位成功，他带球趟入禁区，在门前13米处面对塞萨尔右脚冷静抽射入网，把比分变成了1比0。
这个进球让梅阿查球场沉寂了差不多一秒钟的时间。
现场解说员雷卡很意外的道：“国际米兰刚刚血洗巴萨，不会在保级球队身上输球吧，如果那边罗马输了，这边国际米兰也输了，那穆里尼奥可就玩大了，现场可是有八万多球迷等着看国际米兰夺冠呢，如果你替补能赢球，球迷不会说什么，如果替补输了，哪怕是为了打巴萨做准备，球迷也肯定会有怨言。”
场边的穆里尼奥却面色不变，在教练席上稳稳的坐着。
虽然亚特兰大进球有点让穆里尼奥意外，但他还是相信自己的球员的，高小冬、潘德夫、斯内德都在，后腰上斯坦科维奇和蒙塔里放在其他意甲球队也都是绝对主力，穆里尼奥不相信亚特兰大能翻了天。
经历了欧冠和切尔西、巴萨的数次恶战之后，现在的国际米兰不再像原来那么神经质，面对被亚特兰大领先的形势，球员心态如常，依然稳稳的控制着皮球，向亚特兰大的禁区发起一波又一波的冲击。
比赛第13分钟，比安科对斯内德犯规，主裁判奥尔萨托亮出首张黄牌。
高小冬主罚任意球，他一脚漂亮的落叶斩越过了人墙，但却打在了横梁上弹了出来。
现场一片扼腕叹息之声。
第20分钟，比安科又放倒了高小冬，国际米兰再次获得任意球。
高小冬在门前27米处把任意球传向禁区，中后卫马特拉齐门前七米处头球攻门，但球神奇的被亚特兰大替补门将科波拉扑出。
现场球再次惋惜不已，他们觉得国际米兰今天的运气似乎不怎么好。
第24分钟国际米兰获得了最好的机会！国际米兰后防解围发起快速反击，斯内德中场横敲，高小冬直传，刚刚吃到黄牌的比安科解围失误大腿垫给潘德夫，米利托等球反弹后，在门前23米处右脚凌空吊射，球越过了门将科波拉，但又打在了横梁上。
这下看台上骚动起来，国际米兰球迷有些不安了，他们觉得今天这个比赛有些怪，面对保级球队，他们愣是进不了球，难道34场不败的金身要破在“山寨国米”亚特兰大手里不成？
