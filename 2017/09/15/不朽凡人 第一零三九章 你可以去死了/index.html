不朽凡人 第一零三九章 你可以去死了

发表于 2017-09-15 | 分类于 不朽凡人

午惊德听到莫无忌的话甚至都有些愣神，他还有举例说明的话都没有说出来，对方居然直接说同意了，还让自己提要求。番茄小说网w
本来午惊德是打算让莫无忌先留下来，然后慢慢来推敲莫无忌的来历。现在莫无忌主动提出让自己说要求，午惊德自然是不会错过机会，“流星道友，我也实话实说，道友充实的财力让我决定做这一笔生意。因为这是我黑芥神商最机密的事情，所以道友第一需要告之我们你的来历。”
莫无忌点点头，“这个要求很是合理，我是一个散修。机缘巧合之下，获得了一笔修炼资源。限于我的资质，仅仅是神晶已无法让我更上一层楼，我需要大量好的神丹，还需要一枚好的神格晶。”
午惊德被莫无忌的话是真惊到了，难道这就是一个二货？真以为凭借大量的神晶和神灵脉，就能购买到一堆的修炼资源？这绝对不可能。
可是如果对方真的有来历，说出来岂不是更好？至少黑芥神商不敢动他啊，还是说对方真的是一个散修？
强压住内心的疑惑，午惊德继续说道，“这第二条就是道友需要先交纳一条上品神灵脉，五百万上品神晶作为押金。当然，作为回报，我会让道友观看一下水晶球。识海衍化灵根和灵络我们没有办法用水晶球记录下来，却可以让道友看看别人是如何通过这种衍化手段凝聚神格的。”
“可以，我同意了。”莫无忌毫不犹豫的拿出了一个储物戒指递给午惊德，“这里面有一条上品神灵脉，五百万上品神晶。”
午惊德今天真是彻底的晕菜了，他根本就没有想过莫无忌会如此好说话，这简直是他说什么，莫无忌就同意什么。这让他有些为难了，他本来的意思可不是想要让莫无忌交纳这么多押金，而是等莫无忌不同意，他再说出下一个条件的。
现在不等他说出下一个条件，莫无忌直接同意了，这让他怎么玩？
“怎么？莫非午执事是骗我的？只是为了收我的神晶不成？”见午惊德没有半点反应，莫无忌一皱眉，语气略微有些冷意。
午惊德的确是有些为难，莫无忌的来历是不是散修他还不是非常确定。但是他确定一旦莫无忌看了他黑芥神商的这个水晶影像，那无论如何也不能让莫无忌走出黑芥神商。
他黑芥神商实力低微，这种拥有双紫府和双识海的奇宝一旦泄露，那可是天大的事情。
“自然不会。”听到莫无忌的话，午惊德一咬牙，直接抓出一个水晶球激发说道，“流星道友请观看影像。”
他决定赌一把，如果对方真的是大宗门弟子，那就不应该拿出这么多的神晶和神灵脉来黑芥神商寻找神格晶。他不相信一个大宗门连神格晶也没有。
至于莫无忌是故意上门来找茬的，这根本就被他无视了。莫无忌才什么修为？凭什么来黑芥神商找茬?
影像上一个清晰的身影出现在莫无忌的眼前，莫无忌看见这个身影，差点一拳轰向了午惊德。
这个身影正是天奴，他第一次看见天奴的时候，天奴就非常瘦弱了。此刻的天奴，更是像一个竹竿。两眼无神，皮裹在骨头上，哪里还有半点生息的样子？
在天奴之前，一名身穿锦衣的男子正盯着天奴。尽管天奴识海衍化新的灵络和灵根水晶球上无法表现出来，莫无忌也能猜到，此刻天奴必定是在被迫的衍化灵根和灵络。
见莫无忌有些不平静的样子，午惊德在一边说道，“流星道友，这个双紫府的修士已受不了几次衍化了，所以这次我们收的价格略高一些。需要四条上品神灵脉，三千万上品神晶……”
“好。”水晶影像消散，莫无忌也是吸了口气，语气很是坚决，“我什么时候可以看到？”
见莫无忌连价格都没有还，午惊德立即就知道，莫无忌的身家绝对不止这点。让莫无忌观看天奴衍化灵根和灵络，那就别想了。天奴现在奄奄一息，能不能活下来都是两说。如果不是想要天奴继续为黑芥神商赚神晶，天奴早就被杀了。
午惊德连要求莫无忌签订契约的事情都免了，反正莫无忌这个肥羊他是斩定了。
“这名双识海的修士自然不会在神域新城里面，你需要和我一起过去。”午惊德笑吟吟的说道。
莫无忌脸色一沉，淡淡的说道，“既然如此，那请午执事将东西给我，我去别的商楼看看。”
和午惊德一起离开神域新城，莫无忌还没有那么傻。在神域新城，莫无忌可以动手，因为这里的护阵对他来说是摆设，他可以在这里轻松风移离开。离开神域新城，他才神君一层而已，有什么资格和一个神君四层出去交易？
他是不怕午惊德，同样的他也绝对不相信到时候只会面对午惊德一个人。
“流星道友，这是什么意思?”午惊德一皱眉，语气有些不大好的说道。
东西到他手中，想要让他退出去，那是绝无可能。
莫无忌淡淡说道，“我才不过是育神修为，之所以敢在神域新城交易，那是因为神域新城是我神域最大的神城，这里有巅峰强者坐镇，我交易没有什么危险。一旦离开了神域新城，谁能给我安全上的保证？我只要有神晶，就不相信交易不到神格晶。”
午惊德根本就没有提将东西交给莫无忌的事情，现在听到莫无忌之所以敢来交易是因为这里是神域新城，心里也是大定，嘿嘿一笑说道，“流星道友，押金之所以是押金，那等于是一个口头承诺，道友临时反悔，这是何道理？”
莫无忌一挥手，一堆识海念晶就落在了地上，“午执事，这次是我的不对，但是那神晶和神灵脉却是我购买神格晶的东西，还请午执事还给我。为了表示我的歉意，我这里有一些炼器用的黑神石就送给午执事如何？这些黑神石虽然不是什么高级的东西，却可以炼制出不错的下品神器，甚至偶尔可以炼制出中品神器来。”
说话的时候，莫无忌隐晦的将一枚遁符捏在了手中，他知道自己的这枚遁符绝对不会骗过午惊德的眼神。之所以用遁符给午惊德发现，就是让午惊德不要动手，先稳住自己，将天奴带来。他来这里本来是毁掉黑芥神商这个点的，现在知道天奴在黑芥神商手中，他自然要顺便将天奴带走。
“识海念晶？”午惊德看见这一堆识海念晶，差点惊叫出声，好在他还有理智，没有叫出来。
此刻他心里却绝对不是表面上如此平静，什么炼器用的黑神石？这根本就是识海念晶，同样是可以凝聚神格的宝物。这东西居然有这么一大堆，要值多少神晶？
“哈哈，流星道友多虑了。我黑芥神商做生意凭借的就是信誉，就算是道友不想和我一起过去，我也不会留下道友的押金。道友请稍坐片刻，我立即就让人带那双识海的修士过来。”午惊德赶紧说道，他看见了莫无忌的遁符，也感觉到莫无忌急切的要走。
现在当务之急，就是将莫无忌稳住。就是要干掉莫无忌，在神域也不能光明正大的闹出动静。就因为不知道莫无忌的真实修为，才不敢现在动手。他怕一招之间干不掉隐匿了修为的莫无忌，闹出大动静。
“好，我在这里等着。午执事放心，这种黑神石我有数千万之多，兑换成神晶足够支付给你。”不等午惊德动手抓识海念晶，莫无忌已经是手一卷，将这一堆识海念晶全部收起。
午惊德心里愈发震惊，他心里此刻反而有些担忧起来，莫无忌是不是故意的。否则的话，识海念晶是凝聚神格的，难道他不知道？
这个念头只是在午惊德脑海中转了一下，他就丢在了一边。无论如何，那识海念晶他也不能让莫无忌带走。眼前这个流星再强，在他眼里也是一个蝼蚁。退一万步，哪怕对方真的是大宗门弟子，他杀了后直接逃离而已。有了数千万的识海念晶，他午惊德何必绑在黑芥神商？黑芥神商的死活和他有什么关系？
“我当然相信道友。”午惊德说话间，已经发出了一道讯息。
莫无忌开始闭目沉思，不再说话。事实上是，他的储神络在黑芥神商勾画困杀阵。
见莫无忌不再说话，午惊德有心询问一下莫无忌拿下识海念晶是如何来的，又担心莫无忌怀疑，索性也陪着莫无忌枯坐。
对于可以将天奴带到这里来，午惊德并不担心。天奴几乎等于废掉了，也是他的人在看管着，现在他让自己手下的人将天奴带到神域新城，黑芥神商绝对不会为此来阻拦。
半天时间过去，房间门口的禁制被出动，一个黄发年轻修士出现在宾客室门口。
莫无忌睁开眼睛，午惊德已从那黄发年轻修士手中接过一个灵兽袋说道，“流星道友，你的神念可以渗透进去扫一下，然后我们去静室参悟。”
要杀莫无忌绝对不能在这宾客室，他先将莫无忌骗离这里再说。
莫无忌的神念瞬间就扫到了犹如死人一般的天奴，此刻的天奴全身发灰，比影像上看见的更是可怕，甚至连皮肤都变成了薄薄的一层黑色，发灰发黑的骨骼清晰可见。
莫无忌杀意顿起，他一挥手，所有的困杀阵纹被激发，一指轰向了午惊德，厉声喝道，“你可以去死了。”
