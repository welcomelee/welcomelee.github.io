天道图书馆 第二百五十七章 不是庸医是什么？

发表于 2017-09-15 | 分类于 天道图书馆

“说清阳宗师是庸医？”
“说他是医术低劣的蠢货？”
所有人呆住。
一直对张悬有意见的莫雨小姐，也错愕当场。
直接说一位医道宗师是庸医……这等于辱骂对方的职业素养，彻底撕破脸皮了。
你一个对医道啥都不懂的家伙，装什么大头蒜？
阉掉蛮兽，虽然不多见，却也不是没有过，现在铁齿啸天虎这副模样，不这样做难道还有其他办法？
“张兽师……”
没想到这时候他会说话，而且直接开骂，封堂主也脸色一变，连忙摆手：“清阳宗师，你别生气，他不是故意的……”
“你闭嘴！”
清阳宗师脸色狰狞，像是一个发怒的狂狮，冷冷看向眼前的青年，语气冰寒：“我清阳行医三十九年来，名传王国，医人无数，救死扶伤不知多少人，还是第一次被人这样辱骂。小子，我不管你是不是正式驯兽师，背后有有什么身份，今天不把话说清楚，休怪我翻脸不客气！”
轰隆！
声音结束，身上一股气息直冲云霄，压迫的云涛等人连连后退。
竟然也是以为宗师境强者！
无论何种职业，不光需要特殊天赋，还需要实力与之匹配，达到二星，基本都会是宗师强者，正因为如此，一星职业被称为大师，二星被称为宗师，不光是对职业的尊崇，更重要是对修为的敬畏。
清阳身为整个北武王国最厉害的医师，一生行医，哪被这样辱骂，要不是涵养高，肯定早就炸了。
“没被这样骂……那是因为没遇到我，早遇到，早就骂了！”
面对对方的威压，张悬神色淡然的看过来。
清阳宗师双眉抖动：“你以为我不敢杀你？”
“杀我？没看出病症，就妄下诊断，真要按你所说，非但救不了这头啸天兽，还会害得它当场死亡……这就是你的诊治结果？不是庸医是什么？还杀我……错了不让人说，脸皮咋这么厚呢？”张悬眼皮一抬。
“你……”
不听这话还好，一听之下，清阳宗师气的差点吐血。
他宗师境威压全力释放出来，本以为对方会颤抖的道歉，做梦都没想到不但说的理直气壮，还骂他脸皮厚……
堂堂医道宗师，被人当众辱骂，都快直接疯了。
“是不是觉得不服？那我就跟你说说！”
怒火燃烧，正想发飙，就见眼前的青年，两步来到啸天兽跟前。
“铁齿啸天兽想要看年龄，最直接的根据是牙齿。成年的啸天兽，牙齿裸露在外的一共三尺三寸，不长不短，这点想必只要是驯兽师都知道，不用我多说！”
张悬一指眼前的啸天兽：“这家伙的牙齿，只有两尺七寸，也就是说，距离成年还有一段距离。如果我没看错，它应该才十七岁，按照啸天兽两百年的寿命，只能算是童年……我想问一下，童年的蛮兽，怎么会有交配的**？”
本来封堂主等人也觉得这位张悬是在胡闹，听到这话全都一愣。
这头啸天兽，别人不知道多大，他们几个可是很清楚，从小就养在这里，正好十七年，说的没有一点错误。
如果不是提前知道，光看这家伙如此庞大的体型，绝不敢相信，只有十七岁。
相对于两百年寿命的蛮兽，这个年龄，和人类的七、八岁差不多。
“它是蛮兽……”清阳宗师哼道，话没说完，就被打断。
“不要跟我说，蛮兽的身体和人类不同，发育也不一样，如果这家伙真是发情，想要寻求配偶，也最多燥热难耐，性情暴躁，发个情就会昏迷，沉睡不起……清阳宗师这样判定，难道你觉得……你和它一样？”
张悬接着道。
“对啊！”
“发情最多难受，还没听说谁想女人想的自己昏迷的！”
众人先是一愣，随即嘀咕起来。
成年想要找配偶，这是生理需求，不算什么，不光蛮兽会有，人也会出现这种情况，但从未听说，谁因为找不到女朋友，把自己硬生生憋晕的。
这尼玛完全不符合常理啊！
“哼！”
清阳宗师脸色憋的透红，想要解释，却不知怎么回答，一甩衣袖哼了一声：“你说不是发情，那为何焦躁不安？尤其月圆之夜，来回奔跑，有发泄不完的精力？”
“焦躁不安、有发泄不完的精力，就是想要求偶？这是什么理论？”张悬摇头：“如果只是这个原因，为何不想吃饭，什么都引不起兴趣？”
清阳宗师哑住。
如果真是想要求偶，就算蛮兽不会说，也会有各种表示，不吃饭、对任何事都没兴趣，的确让人费解。
“根据刚才那个学徒的语言，我们可以知道，这家伙十天前，只是不想吃东西，这几天逐渐加重，甚至于到今天早上，再也承受不住而昏迷。如果真是想择偶的话，直接飞走即可，貌似兽堂也没用铁链拴住它，禁止它飞行吧！”
众人同时点头。
兽堂巴结都来不及，怎么可能拴住这家伙？
真想择偶，真要难受，可以自己飞去出去找啊，不至于一直在院子里待着硬生生把自己憋昏吧！
听到这些分析，所有人都对之前清阳宗师的诊断有了疑惑，一个个满是古怪的看过来。
“那……你说是怎么回事？”
清阳宗师脸色一白，也发现忽略了不少东西，没了刚才的怒气，忍不住问道。
“不是求偶，剩下的就简单了……那就是，啸天兽真的身患重病或者有其他原因！”张悬环顾一周：“大家可以看看啸天兽的蹄爪。”
“啸天兽和鹰类的蛮兽不同，有厚厚的肌肉包裹，与人类的脚掌类似，这个爪子，按照正常情况，应该是金青色，而眼前这头，肉色中透露着红色纹路。”
众人齐刷刷向啸天兽的蹄爪看了过去，一看之下，果然发现了一些血红色的纹路。
这些纹路极为细小，和毛细血管相似，不仔细看，难以发觉。
“一般蛮兽都有肉冠，这头啸天兽也不例外，正常情况下的肉冠，带有血红色，而此刻却微微泛白，虽然细微，却也能够看到。”张悬继续道。
众人再次看去，果然如他所说。
“这到底是什么病症？”
见他说出这么多细微的变化，众人再次相信了几分，封堂主忍不住问道。
“说病不是病，仔细说起来，我还要恭喜封堂主。”
张悬笑了笑。
“恭喜？恭喜我做什么？”
封堂主一脸发懵。
啸天兽昏迷不醒，也不知有没有生命危险，再说，就算知道了病症，知道如何救治，也要救醒再说，这时恭喜做什么？
“如果我没看错，啸天兽应该血脉进迁了！”张悬道。
“血脉进迁？这……这怎么可能？”
封堂主愣住，其他人也不敢相信。
蛮兽的血脉进迁，和变异类似，很难遇到，没有最根本的诱因，是不可能发生的。
所谓的诱因，就是吸收了更精纯的血液，或者吞服了某些天材地宝……
这些貌似都没有过！
而且，就算血脉进迁，也不至于昏迷，跟病重一样吧。
见众人不信，张悬轻轻一笑看向封堂主：“半个月前，这头铁齿啸天兽，是否去过阴气浓郁之地？”
“这……半个月前，它曾专门送人去了阴风谷！”回答的是王兽师。
正是他驾驭送过去的，记得很清楚。
“阴风谷阴气浓郁，盛产一种厉害的阴属性果实，阴风果，可帮助阴属性体质的人，修为大增。想必那人就是以此为报酬的吧！”张悬道。
“是！”王兽师点头。
搭乘兽堂的兽乘，可以给钱，也可以给一些稀有的天材地宝。
阴风果名气极大，单独出售的话，价值不菲，而且有价无市，对方以此为报酬，王兽师自然答应。
“原因就出现在这枚阴风果上！”
张悬笑了笑：“这头啸天兽的性别，想必大家都知道，雄性。体内阳火卓盛，服用了阴风果，阴阳相冲，按照道理，最多不太舒服，以它如此巨大的体型，很快就能抗衡过去！结果，你们偏偏给它服用了清神水。”
“清神水，静心清神，可以消除暴躁，其中最主要的成分是【酵母草】。”
“这是平稳中和性的药物，按理说服用再多都没事，不过，不巧的是，这东西恰好和阴风果属性相冲，一旦接触，会立刻引发力量，让阴属性气息彻底爆发，激发出三倍以上的效果，两者结合，正是炼制【破阳丹】的主要材料，对阳属性体质的生命有极大的激活作用。”
张悬继续道：“啸天兽体内的阳属性被激活，这才夜间不眠，拥有发泄不完的精力，如同si春！”
“如果只是这样，它依旧能坚持过去，只要体内阴阳平衡，同样能解除隐患，可惜……封堂主又让人给吃了两枚【去热丹】。”
说到这张悬满是怜悯的摇了摇头：“去热丹之中蕴含极寒草，这东西能够消除体内的阳火……啸天兽刚刚激**内的阳属性力量，就拿这东西给它吃，还一吃两枚，相当于直接在火苗上扔下冰块……一冷一热之下，没当场死亡，就算命大了！”
说实话，这头啸天兽还真够可怜的，本来没事，差点硬生生被他们给喂死。
还救治……简直就是杀人！
封堂主、卢兽师等人脸色难看。
很显然，他们也没想到，啸天兽之所以这副模样，竟然是因为他们……喂错了药！
“你们仔细回忆一下，是不是因为喂了药之后，这头啸天兽的病情才逐渐恶化？”张悬接着问道。
“是……”回想了一下刚才那位学徒说的话，众人忍不住点头。
的确如此。
谁都没想到，正常喂出的药物，竟然成了起了反作用。
“体内阳属性激活，与寒属性对碰，此时，如果将它阉掉，阳火消退，阴气上升，用不了一个时辰，它就会断绝呼吸，当场死亡！”
说到这，张悬看向不远处的清阳宗师：“这是我为何要说你是庸医的原因。真要按你说的做了，非但救不了啸天兽，恐怕现在已经是一具尸体。”
“这……”
清阳宗师浑身冰冷，做为二星医师，对医道有极高深的了解，对方说的丝丝入扣，环环递进，分析的更是鞭辟入里，没有任何错误，如同亲眼所见一般，就连他都佩服不已，想反驳，都找不出理由。
“你刚才说的血脉进迁，是怎么回事？”
怕清阳宗师难堪，封堂主急忙转移话题。
“体内阳属性被激活，稳定下来，自然血脉进迁，反之，则会死亡。也就是说，现在的啸天兽，能扛过去，修为会有进步，得到极大好处；抗不过去，就会死亡。”
“那……可有救治方法？”封堂主满是紧张。
“当然有，不过……可能需要一位炼丹师的配合。”张悬迟疑了一下道。
“莫雨小姐就是正式炼丹师，她可以帮忙。”一侧的云涛急忙道。
众人齐刷刷将目光集中过来。
“要怎么做，我可以出手。”莫雨小姐双手背在身后，一脸傲然。
虽然对眼前这个家伙不太感冒，但只要能救啸天兽，让她做什么事都可以。
再说，其他方面都比不上对方，这次终于可以展露炼丹师天赋扬眉吐气，一雪前耻了。
你不是驯兽厉害吗？不是名师学徒吗？不是连医道都能说出一二三吗？
虽然这些都比不上你，我至少还有一个丹道！
至少我是一位正式的一星炼丹师！
一脸骄傲的走出人群，正想问对方如何帮忙，就见对面的青年眉头一皱，带着质疑的语气问道。
“你……会炼丹吗？”
“你……”
莫雨小姐一个趔趄，差点没当场炸了。
你还会不会说人话？
我是一星炼丹师，当然会炼丹了！
不会炼丹怎么考上的？
早知道这家伙不会说啥好听的，没想到如此可恶。
牙齿咬的咯咯作响，气的呼吸急促，胸口不停起伏：“你见过不会炼丹的炼丹师吗？”
“见过……”张悬摇头：“我怕你也是辩丹成就的炼丹师，这样就不会炼制……”
“辩丹？那可是对炼丹精通到一定实力的丹道大师才能做到的！”
莫雨小姐眼中露出崇拜之色：“这种前辈，每一个都有惊天动地之才，你居然说他们不会炼制丹药？开什么玩笑！如果你敢在炼丹师公会这样说，绝对会被人当场打死……”
她的话还没说完，就见对面的青年默默取出一个炼丹师的徽章戴在身上，挠挠头，一脸尴尬。
“我就是你说的那种前辈，刚好不会炼丹……”
