天道图书馆 第一百九十章 陆寻借钱【第一更】

发表于 2017-09-15 | 分类于 天道图书馆

不光他如丧考妣的表情，一侧的刘凌、庄贤也嘴巴张开，仿佛能吞下整个鸭蛋。
武者八重宗师境，和通玄境有着质的差别，生命层次也会不同，寿命更是因此延长。
普通武者的寿命，七、八十岁就是极限，而达到宗师境，活过一百都不是问题。
正因为如此，想要突破和天堑一样困难。
从他们这些名师，都困在半步宗师，没办法成功就可以看出难度。
沈洪的情况，他们一来到天玄王国就专门观察了，生机破灭，随时都会死亡，就算真气足够，也根本不可能有精力突破，做梦都没想到，在他们眼中最难得一关，居然被眼前这位杨师，一瓶药轻松解决。
剧烈的反差，让几人觉得自己之前的认知，全部崩塌。
沈追陛下和三师一样的想法，觉得眼前这位杨玄，肯定是故意整他，怎么都没想到，老祖竟然真的成功了！
这可是实打实的宗师境……
“多谢杨师……”
过了不知多久，沈洪猛地站起，一声长啸，声如巨龙，几步来到张悬面前，直接跪倒在地。
困在原地这么多年，本以为必死无疑，再无机会，做梦都没想到，竟然成功了！
见他跪在面前，张悬也不去扶，双手背在身后，环顾一周，缓缓踱步。
“沈洪是寿命到了，身体机能衰退到极限，此时，对活下去的**也没那么强烈了，正因为如此，他才能看穿生死，觉得死亡也没那么可怕。这种情况下，无论心境还是身体，都达到了衰败期，别说补药，就算厉害的二品、三品丹药，都不可能让其突破最大的难关，冲击宗师！”
众人知道他实在解释刚才的所作所为，全都大气不敢出，仔细聆听。
“所以，想要让他继续活下去，首先要给他活下去的理由，而且还是不能拒绝的理由！”
“只要是男人，任何时候都无法拒绝年轻的女人，更何况还吃下药，药物刺激下，全身机能肯定会重新燃烧，恢复青春……这种情绪一旦积累足够，想要再突破，也就会容易许多。”
“当然，最重要的是他修炼的功法，如果我没看错，应该是烈阳决！”
说到这张悬看过来。
“是！”沈洪点头承认。
他修炼正是烈阳决，天玄王国最厉害的功法。
“这种功法，烈日炎炎，阳气冲天，是不可多得的秘籍，男人修炼，威力更胜！按照常理，他的身体机能丧失干净，就算有药辅助，也不可能重回巅峰，但多年修炼烈阳决，体内积攒了无数的阳气真火，此刻经过药物引发，再加上美女勾引，就好像火星仍在了油桶上，立刻点燃了熊熊大火。”
“全身的精气神集中在一点，突破起来，想不突破都难！”
解释完，张悬安静站在原地，周围鸦雀无声。
其实他最重要的没说出来。
无论药还是烈阳决，都只是突破的一个点，真正事关重要的……是他将对方敲晕。
每动手一次，就将自己的真气注入对方体内，打通一道经脉堵塞桎梏之处，经脉彻底畅通，再加上情绪激发到了极点，这才一举成功，踏出最后一步。
当然，他真气太过精纯，实力也只有辟穴境，忽悠孙强等人很简单，真要让沈洪意识到，肯定能看出些什么，所以，才故意让沈追陛下说是他打昏了对方。
并非故意装好人。
至于现在，对方突破成功，自己的真气也和对方的彻底融合，就好像清水融入浑水之中，再想寻找，也找不到了。
“这……”
听完解释，三大名师一个个呆若木鸡。
这尼玛……也行？
他们都是科班出身的名师，见过不少帮人突破的手段，可这种突破方式……还是第一次看到。
这已经不是离经叛道了，简直就是不可思议。
打破头也想不出来。
吃药吃的突破……想想也是醉了。
“突破有很多种，一直墨守成规，很多情况是行不通的，做为名师，需要不停琢磨，而不是恪守规则。”
张悬摆了摆手，修长的影子被拉的长长，整个人像是不被人理解的超级强者，说不出的寂寞。
“杨师教训的是！”
三大名师同时鞠躬。
此刻他们对这位杨玄，佩服的五体投地。
不佩服不行，太神了！
一顿乱棍，就让多年无法突破的庒师突破成功，随便配制出毒师才能配出的药液，吃药吃的沈洪突破……
即使亲眼所见，都觉得如同做梦。
“嗯！”见他们理解，张悬点点头，正打算告辞回府邸，就见一双眼睛出现在眼前，一脸的羞涩和兴奋。
后退了一步，这才看清，正是三师中的郑非。
刚才他质疑的最狠，甚至开口保证，只要沈洪能够突破，头割下来，现在却双眼放光，和中了大奖一样。
“你要干什么？”张悬满是警惕。
“杨师，我也困在半步宗师境好多年了，现在就要准备药和美女，也想冲击一下宗师境……”
郑非手里不知何时拿着一个玉瓶，倒出里面的丹药，就要吞下去。
“……”张悬。
“……”沈追陛下、刘凌、庄贤。
好不容易劝阻住对方，张悬无奈。
每个人修炼的功法不同，性格不同，修为不同，突破方法肯定也是不一样的，做为名师郑非肯定懂这个道理，但他还是小看宗师境界的诱惑力了。
寿命大大增加……单这一条，就能让无数人失去理智和判断力。
其实不光郑非意动就连刘凌、庄贤看向自己的目光也变得火热。
连快要死亡的沈洪，这位杨师都能指点的突破，他们精力旺盛，又是名师岂不更容易突破？
想到这点，三位名师各自打好了注意，打算跟眼前这位搞好关系，争取能让他帮忙出手指点一次。
见他们都想让自己指点，张悬看的头皮发麻，正打算告辞，就见大殿外面一个护卫走了进来。
“回禀陛下，洪天学院陆寻、王超两位老师在殿外求见！”
“这时候过来干什么？”沈追陛下一愣，随即点头：“让他们进来吧！”
这两位明星教师，极有可能成为以后的名师，就算他是国王，也愿意轻易得罪，更何况和二人的长辈，又有着极深的渊源。
一侧张悬也是一愣。
“陆寻？王超？”
这二人刚才不是要拜访自己？怎么跑这里来了？
难不成离开府邸的时候被发现，追了过来？
不可能啊……
心中疑惑，想要离开的脚步停了下来。
陆寻要和他师者评测，之前从未见过，既然碰上，刚好也看看，到底长得啥样，又有什么本领。
沈追陛下让诸多舞姬退了出去，重新收拾了一下大厅，沈洪因为衣服全部撕烂，不方便见人，也跟着离开。
几人刚刚坐定，就见两个青年大步走了进来。
看清楚二人长相，张悬眉头一皱。
不正是昨天在校园门口遇到，一脸高傲、不屑一顾的两个二货吗？
闹了半天，他们就是传说中的陆寻、王超啊！
难怪眼神中带着鄙视，恐怕已经认出自己是谁了。
“陆寻（王超）见过陛下，见过刘师、庄师、郑师！”
二人同时抱拳。
沈追陛下他们早就见过，自然认识，三师虽然没见过，却穿着名师的服饰，很容易分辨，至于一侧的张悬，从穿着上看，锦帽貂裘，举止浮夸，一看就是暴发户……只瞥了一眼，就不再理会。
明星教师也有明星教师的骄傲，不是任何人都要行礼的。
他们一直听说杨师想要拜访，实际上从未见过人，也就不知道坐在眼前这位就是。
“两位这时候来找我，所为何事？”
沈追陛下见二人不理会杨师，有些不悦，想要介绍，就见后者示意不用，只好把话咽在口中，疑惑的问道。
“我俩冒昧前来，的确有个不情之请。”陆寻犹豫了一下，开口道：“想向陛下借金币三百万……”
“借钱？”
沈追陛下奇怪。
凭借二人的家世，还用向自己开口？
而且，开口三百万，借这么多钱要干什么？
“我们二人想要拜访杨师，想必陛下也听到了规矩，进门就需要缴纳三百万！”见他疑惑，陆寻也没遮掩，将自己的目的说了出来：“我们这钱只是暂借，只要能见到杨师，我有自信能让他收为学徒，到时候三百万肯定也会被退换回来，归还陛下了。”
“要拜见杨师？”
听到他的目的，沈追陛下、三师对望一眼。
你要拜见的杨师就坐在不远处，可你自认为高傲，连个礼节都不行一下，熟视无睹……
就这样还想做他学徒……
众人脸色古怪，一侧的张悬挠挠头看了过来，他真有些奇怪，这家伙哪来的自信，三百万会被退回来？
“你就这么确定杨师会收你？万一不收，这三百万退不回来，你拿什么偿还？”
伸手阻止住沈追陛下和三师想要说出自己身份的举动，张悬忍不住看了过来。
