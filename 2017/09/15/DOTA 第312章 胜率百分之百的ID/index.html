DOTA 第312章 胜率百分之百的ID

发表于 2017-09-15 | 分类于 DOTA

慕容在跟送分狂魔这个家伙加了好友以后，那家伙就心满意足的下线了，慕容想起来一件重要的是要做，他打开vs平台，搜索了一下尘埃这个人的id，他还没有上线。
看来还真的是掉线了啊，估计明天就会上线了吧？慕容心里这么想道。
慕容又看了一下这个家伙的信息，里面其他的信息都是空的，只有一项让慕容感觉到有点不可思议。
这个家伙目前的vs等级是18级，级别虽然不是很高，但是有一点就是，他在这18级之中只输过一局，胜率是99%，慕容知道他输的这一局就是刚刚打得这一局，还是因为掉线导致的，如果不掉线的话，慕容也不知道自己能不能赢得这局比赛，很有可能这个尘埃会继续赢下去。
看到这个慕容感觉有点不可思议了，这个家伙也太强悍了吧？要知道，在等级别低的时候，你可能凭借自己的个人实力打爆对方，一个人拯救世界，但是在10级左右的这个菜鸟段之内，水可深着呢。
任何一个dotaer都知道，菜鸟房的水很深，往往比vs一房都可怕，因为这里充斥着各种各样的选手，有真正的菜鸟，就是那种连补刀是什么意思都不知道的纯新手，还有那种半瓶子水的选手，这种选手往？一？本？读？小说 …往以为自己对dota的理解就是正确的，打法专断，自以为是，还有一种人就是一些真正的大神在玩小号，或者在带小号，比如带妹子啊，带同学啊之类的。
很有可能，你队伍里有四个真正的菜鸟，那种你看一眼他们的走位就想冲上去给他们几下的人，而对面的队伍里却有几个黑店或者几个高手，你想赢？只能说很难很难。
但是这个尘埃，他的胜率居然是百分之百，排除掉这次掉线的意外，这是什么样的实力？平心而论，让慕容重新练一个小号的话，他也没有自信能够一次不输的打到18级。
正在慕容震惊的时候，刚好sony从身边经过，看到了慕容正在盯着屏幕看一个vs账号信息，而且满脸的凝重，不由得好奇的停在了慕容身边问道：“干嘛呢？什么东西这么好看？”
“sony，你看看这个……”慕容没有明说什么，而是让他自己去看。
起初sony是面带着不在意的笑容凑过去看的，当他看完的时候，脸色也变得很凝重，他连忙问道：“这是你朋友？”
“不是，刚才打得一局的对手。”慕容回道。
sony看到输了一局，不由得说道：“我擦，这小子胜率这么高，这唯一输的一局是刚才被你给终结的？”
sony问这话的时候，他也很震惊，因为他自己也玩vs，他自己的账号胜率都没有这么高，虽然自己的级别很高，但是他知道自己从1级打到26级的时候，在18级之前可输了好多。
“不是被我们终结的，刚才他掉线了，如果不掉的话，我自认为自己不一定能打得过他。”慕容非常认真的说道。
“掉线的?那么说，他的胜率是百分之百？”sony彻底震惊了，震惊之余，他都没注意控制自己的情绪，大叫了起来。
“sony，你那在咋呼什么呢？我还在跟妹子聊天呢，好好的都被你给破坏了氛围。”万宝路不满的说道。
他们刚好打完了一场训练赛，然后每个人自由活动一小时，到了晚上12点的时候，再练1个bo3，这就是他们的训练方式。
“万b，你过来看看，你自己过来看看。”sony非常急切的呼叫他的好基友万宝路。
“什么事啊？这么大惊小怪，有点深度好不好，男人要有气度。”万宝路说完放下耳机，然后慢慢的走了过来。
爬下来冲着屏幕只看一眼，他就被吸引住了，因为他看到了那个胜率99%的数字，这个数字是如此的刺眼，让他的目光久久不能移开。
“如果我告诉你，这唯一输的一把是刚才掉线导致的，你会是什么反应？”sony像是看热闹的一样看着万宝路。
“什么？掉线？不是输的？怎么可能？刷的吧？”万宝路一脸的不相信。
“你刷一个给我看看。”sony反驳道。
“龙哥，龙哥，你来看看这个。”万宝路立马想起来了龙浩，把龙浩叫了过来。
周峰和林灿也围了过来，几个人看到这个战绩之后，都非常的震惊，这个家伙如果真的不是刷的话，那他的实力也太恐怖了。
“能联系得到他不？正好你们的二队还少一个人呢。”龙浩问道。
慕容看到龙浩问道，也只能无奈的摇了摇头说道：“龙哥，他自从掉了之后，就一直没有上线，而且我们打得时候，别人跟他说话，他也从来不回，也不说话，非常的神秘。”
龙浩听完之后，又望了一眼那个非常惹眼的战绩，说道：“往后，你多关注一下他，说不定哪天他就上线了呢！”
“好的，我会每天都看看他上线没？如果上线，我就想尽办法把他拉到我们这里来。”慕容说道。
“嗯！”
几个人又讨论了一会这家伙的战绩之后，就各自散了，大家也都猜测过是哪个职业大神的小号，但是慕容描述了一下他的小黑的打法之后，还真的在当今的职业圈里面找到一个这种打法风格的小黑来。
最后一致的观点认为，这很有可能，是一个dota新人，但绝对是小号，可能技术练到成之后，新练的一个号。
往后的几天里，慕容每天都会登录vs平台，搜索一下尘埃这个id，但是让他失望的是，在往后的几天里，尘埃再也没有上线过。
每天慕容登录上去，看到的都是那个灰暗的角色名，未在线。
难道这家伙的账号用一次就不用了？还是说他发生了什么意外？
否则不可能再也不上啊！
这不免让慕容产生了深深的遗憾，遗憾的不仅仅是无法联系到这个人，还有就是无法再跟这么犀利的人当对手了。
人生得一个值得欣赏的对手，那可是一件非常幸运的事情！
